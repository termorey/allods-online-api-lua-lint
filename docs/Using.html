Пользовательские дополнения

Общее описание

Игра "Аллоды Онлайн" позволяет игрокам создавать и использовать собственные дополнения ("моды", "аддоны") пользовательского интерфейса. Дополнения поддерживаются только для режима "внутри игры". Т.е. нельзя изменить функциональность, например, главного меню игры или экранов создания и выбора персонажа. Пользовательские дополнения будут работать только после выбора персонажа и захода им в игру.

Управление пользовательскими дополнениями

При наличии установленных пользовательских дополнений в главном меню игры появляется кнопка "Включить/выключить пользовательские дополнения". По умолчанию дополнения отключены, т.е. при заходе в игру будут работать только оригинальные интерфейсы игры. Состояние разрешения работы пользовательских дополнений сохраняется между запусками клиента игры. При изменении версии игры (установке ее обновления) дополнения автоматически выключаются.

Замечание: указанное выше состояние "выключить пользовательские дополнение" отключает сразу все пользовательские дополнения. При этом никакое их использование невозможно.

Для отключения отдельных пользовательских дополнений можно использовать следущие способы:

  • Сделать дополнение недоступным для игры - см. пункт "Установка пользовательских дополнений".
  • Отключить автоматический запуск дополнения при заходе в игру - см. раздел "Создание пользовательских дополнений".
  • Отключить дополнение из скрипта (из другого дополнения) - см. раздел "Создание пользовательских дополнений".

Установка пользовательских дополнений

Все данные, относящиеся к пользовательским дополнениям, должны находиться в папке data\Mods.

Собственно дополнения должны находиться в папке data\Mods\Addons. Каждое дополнение - это отдельная папка внутри data\Mods\Addons. Для установки пользовательского дополнения  необходимо скопировать его в эту папку.

Часть дополнений может требовать для своей работы наличия других дополнений или каких-либо общих элементов. Общие элементы должны находиться внутри папки data\Mods, но вне папки непосредственно с дополнениями (data\Mods\Addons). Более точные требования определяются конкретным дополнением.

Чтобы сделать дополнение недоступным для игры, необходимо удалить папку с ним из data\Mods\Addons. Можно либо физически удалить папку, либо перенести ее в другое место (например, в data\Mods).

Упакованные пользовательские дополнения

Более удобный способ распространения пользовательских дополнений - в виде архивов. Такие архивы должны иметь расширение *.pak и помещаться в ту же папку, что и "развернутые" - data\Mods\Addons. Например:

data\Mods\Addons\UserAddon01.pak
data\Mods\Addons\UserAddon02.pak

Локализованные пользовательские дополнения

Многоязыковые дополнения, помимо основных файлов или архива, могут включать в себя еще и архивы с локализацией на разные языки. Архивы с локализацией имеют расширения *.rus, *.eng и т.д. Их следует помещать в ту же папку, что и основное дополнение или его архив - data\Mods\Addons. Например:

data\Mods\Addons\UserAddon01.pak - основное дополнение в виде архива
data\Mods\Addons\UserAddon01.rus - локализация на русский
data\Mods\Addons\UserAddon01.eng - локализация на английский
data\Mods\Addons\UserAddon01.fra - локализация на французкий

Нужная локализация будет определяться по языку, с которым запущен клиент. (Выбранный язык можно посмотреть в файле Personal/global.cfg, переменная localization.)
