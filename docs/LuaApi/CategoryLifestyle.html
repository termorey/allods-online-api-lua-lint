CategoryLifestyle

События и функции Lua API, относящиеся к работе со стилем главного игрока.

У главного игрока имеется несколько пресетов одежды, которые он может произвольно преключать. В пресет стиля входит несколько особых типов (dressSlot) одежды. При переключении стиля игрок переодевает необходимые предметы. Список предметов в активном пресете автоматически изменяется при переодевании предметов обычным способом.
