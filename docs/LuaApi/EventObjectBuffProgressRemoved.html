EVENT_OBJECT_BUFF_PROGRESS_REMOVED

Событие посылается каждый раз, когда у объекта пропадает баф для которого требуется визуализация прогресса.

-- поля:
objectId: ObjectId - идентификатор объекта на который повесили баф
buffName: WString - имя бафа
buffId: ObjectId - идентификатор бафа
sysName: String - системное название бафа
