Пользовательские дополнения

На главную

options.GetOptionByCustomId( sysCustomId )

Возвращает идентификатор пользовательской опции по ее текстовому идентификатору из базы.

-- библиотека:
options

-- объявление:
function GetOptionByCustomId( sysCustomId )

-- параметры:
sysCustomId: string - текстовый идентификатор опции

-- возвращаемые значения:
ObjectId or nil - идентификатор опции или nil, если опция не существует

-- пример:
local optionId = options.GetOptionByCustomId( sysCustomId )
