avatar.GetVariables()

У игрока могут быть некоторые гейммеханические переменные (например, количество Сил Льда у мага). Этот метод возвращает список идентификаторов этих переменных.

Доступные для просмотра переменные указаны в "\Client\GMUIHelper\VariablesAliases.(VariablesAliases).xdb"

-- библиотека:
avatar

-- объявление:
function GetVariables()

-- параметры:
нет

-- возвращаемые значения:
table of VariableId - индексированная с 0 таблица с идентификаторами ресурсов имеющихся переменных

-- пример:
local ids = avatar.GetVariables()
if ids and ids[0] then
  local info = avatar.GetGetVariableInfo( ids[0] )
  if info then
    local currentValue = info.value
  end
end
