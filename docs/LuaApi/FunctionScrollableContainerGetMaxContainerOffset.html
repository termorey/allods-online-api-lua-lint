GetMaxContainerOffset( self )

Возвращает максимальное значение на которое может быть прокручен контейнер

-- класс
ScrollableContainerSafe

-- семантика
function GetMaxContainerOffset( self )

-- входные параметры:
нет

-- возвращаемое значение:
number (float) - максимальное значение прокрутки

-- пример:

-- store max offset
local maxOffset = wtScrollableContainer:GetMaxContainerOffset()
-- прокрутка контейнера до середины
wtScrollableContainer:SetContainerOffset( maxOffset / 2 )
