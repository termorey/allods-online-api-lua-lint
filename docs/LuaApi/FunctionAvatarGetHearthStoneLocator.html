avatar.GetHearthStoneLocator()

Возвращает информацию о месте в которое ведет HearthStone (карта, зона, локатор).

-- библиотека:
avatar

-- объявление:
function GetHearthStoneLocator()

-- параметры:
нет

-- возвращаемые значения
map: WString - имя карты
zone: WString - имя зоны
locator: WString - имя локатора
locationId: ObjectId - идентификатор точки телепортации (для использования при вызове avatar.UseHearthStoneToLocation( locationId ))

-- пример:
local locatorInfo = avatar.GetHearthStoneLocator()

См. также avatar.UseHearthStoneToLocation().
