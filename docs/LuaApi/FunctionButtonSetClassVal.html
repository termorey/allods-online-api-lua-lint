ButtonSafe:SetClassVal( self, tag, value )

Задает значение для подстановки класса (из тега "rs") для данного ButtonSafe.

Замечание: в текущей реализации нет проверки, что данное значение не будет использоваться для подстановки значений (в теге "r").

-- класс
ButtonSafe

-- семантика
function SetClassVal( self, tag, value )

-- входные параметры
tag: string/WString - тег, для которого задается значение
value: string/WString - значение тега

-- возвращаемое значение отсутствует

-- пример
wtButton:SetClassVal( "button_style", "button_red" )
