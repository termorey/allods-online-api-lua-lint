На главную

avatar.DistributeStatPoints()

Окончательно распределяет пункты характеристик, предварительно запланированные к распределению функцией avatar.ImproveInnateStat( stat, amount ). Работает только если avatar.CanImproveInnateStats() даёт true, при этом переключает это состояние в false до ответа сервера.

При любом ответе сервера переключает avatar.CanImproveInnateStats() обратно в true (если нет дополнительных обстоятельств). При удачном завершении операции через некоторое время произойдёт изменение данных, выдаваемых avatar.GetInnateStats(), с событием EVENT_AVATAR_STATS_CHANGED).

-- библиотека:
avatar

-- объявление:
DistributeStatPoints()

-- параметры:
нет

-- возвращаемые значения:
нет



-- пример:
avatar.DistributeStatPoints()
