CategoryDepositeBox

BlurpSearch: CategoryLuaApi CategoryDepositeBox

Функции Lua API библиотеки depositeBox. Библиотека предназначена для работы с внутриигровым хранилищем предметов игрока ("банком").

Замечание. Для задания внешнего вида сумок-хранилищ, включая распределение слотов, предлагается использовать ту же схему, что и для обычных: использовать sysName предмета (сумки-хранилища).
