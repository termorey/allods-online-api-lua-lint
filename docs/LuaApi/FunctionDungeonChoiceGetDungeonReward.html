На главную

dungeonChoice.GetDungeonReward( dungeonIndex )

Возвращает список наград, доступных для получения при атаке личного приключения.

-- библиотека:
dungeonChoice

-- объявление:
function GetDungeonReward( dungeonIndex )

-- параметры:
dungeonIndex: number (integer) - индекс личного приключения в списке

-- возвращаемое значение:
table of table - индексированный с 0 список таблиц, описывающих отдельную награду, с полями:
 index: number (integer) - индекс этой награды в списке
 name: WString - название награды
 count: number (integer) - количество выдаваемой награды

-- пример:
local list = dungeonChoice.GetList()
if list[ 0 ] then
  local rewards = dungeonChoice.GetDungeonReward( dungeonIndex )
  if rewards[ 0 ] then
    LogInfo( rewards[ 0 ].name )
  end
end
