Пользовательские дополнения

На главную

dungeonChoice.GetList()

Возвращает список личных приключений на выбор для атаки.

-- библиотека:
dungeonChoice

-- объявление:
function GetList()

-- параметры:
нет

-- возвращаемое значение:
table of table - индексированный с 0 список таблиц с полями:
 index: number (integer) - индекс этого приключения в списке
 categoryName: WString - название категории личного приключения
 hasRooms: boolean - если true, то нужно будет сражаться
 level: number (integer) или nil - nil если нет информации; иначе уровень сложности категории приключения

-- пример:

local list = dungeonChoice.GetList()
if list[ 0 ] then
  LogInfo( list[ 0 ].ownerName )
end
