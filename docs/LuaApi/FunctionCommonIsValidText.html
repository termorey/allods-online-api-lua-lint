common.IsValidText( text )

Проверяет что переданный аргумент является валидным текстом. Валидными считаются ValuedObject, непустой ValuedText и непустой WString.

-- библиотека:
сommon

-- объявление:
function IsValidText( text )

-- параметры:
text: any value - проверяемое значение

-- возвращаемые значения:
boolean - true если переданный аргумент является валидным текстом

-- примеры:
local isValid = common.IsValidText( text )
