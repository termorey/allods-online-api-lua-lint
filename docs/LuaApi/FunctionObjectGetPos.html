На главную

object.GetPos( objectId )

Возвращает координаты какого-либо объекта, если разрешено их получать. В данный момент разрешено получать координаты транспортов и детектируемых юнитов.

-- библиотека:
object

-- объявление:
function GetPos( objectId )

-- параметры:
objectId: ObjectId (not nil) - идентификатор объекта

-- возвращаемые значения:
GamePosition or nil - позиция объекта, если он найден и можно получать его позицию

-- пример:
local pos = object.GetPos( objectId )
if pos then
  local x = pos.posX
end
