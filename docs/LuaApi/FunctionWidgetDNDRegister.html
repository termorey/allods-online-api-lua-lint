DNDRegister( id, isDragOnly )

Зарегистрировать виджет для drag&drop. Регистрирует указанный виджет и уникальный идентификатор для него. Правила назначения идентификаторов смотри в общем описании DnD системы. Если нужен какой-нибудь идентификатор (без совместного использования), то генерировать его вручную не следует, надлежит использовать DNDRegisterGeneric

Можно указать в каком режиме для данного контрола может начинаться перетаскивание (drag attempt) - либо только на перетаскивание, либо как на перетаскивание, так и на клик.

-- класс
метод доступен для всех типов виджетов

-- семантика
function DNDRegister( self, id, isDragOnly )

-- параметры:
self: WidgetSafe - виджет, для которого будет обрабатываться drag&drop
id: number (int) - уникальный идентификатор, связанный с виджетом wtSrc
isDragOnly: boolean - drag&drop  включается либо только на перетаскивание, либо как на перетаскивание, так и на клик

-- возвращаемые значения:
нет

-- примеры:
wtPanel:DNDRegister( id, false )
