Пользовательские дополнения

На главную

astralUnit.HasAggro( unitId )

Возвращает true, если астральный моб агрессивен к кораблю, на котором находится главный игрок.

См. также EVENT_ASTRAL_UNIT_AGGRO_CHANGED

-- библиотека:
astralUnit

-- объявление:
function HasAggro( unitId )

-- параметры:
unitId: ObjectId (not nil) - идентификатор юнита

-- возвращаемые значения:
bool - true, если агрессивен

Пример:

local isAstralUnitInCombatWithMainPlayer = astralUnit.HasAggro( unitId )
