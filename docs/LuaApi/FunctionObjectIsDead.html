Пользовательские дополнения

На главную

object.IsDead( objectId )

Возвращает информацию о статусе смерти интерактивного объекта (мертв да/нет).

Для игроков означает, что игрок мертв и лежит на месте смерти (не в чистилище).

-- библиотека:
object

-- объявление:
function IsDead( objectId )

-- параметры:
objectId: ObjectId (not nil) -- идентификатор объекта

-- возвращаемые значения:
boolean - true, если объект мертв

-- пример:
object.IsDead( objectId )
