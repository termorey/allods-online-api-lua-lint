Пользовательские дополнения

На главную

bulletinBoard.ReadSection( postTypeId )

Возвращает информацию о содержимом указанного раздела доски объявлений. Если информации еще нет на клиенте, то отправляется запрос на сервер и возвращается nil. При получении информации приходит событие об изменении доски объявлений.

-- библиотека:
bulletinBoard

-- объявление:
function ReadSection( postTypeId )

-- параметры:
postTypeId: PostTypeId - идентификатор раздела доски объявлений

-- возвращаемые значения:
таблица со списком объявлений или nil, если информация недоступна, таблица индексирована [0..], значения таблицы:
 number (int) - идентификатор объявления

-- примеры:
local postIds = bulletinBoard.ReadSection( postTypeId )
if postIds then
 for i = 0, GetTableSize( postIds ) - 1 do
  local post = bulletinBoard.GetPost( postIds[i] )
 end
end
