depositeBox.IsMoveModified( depositeSlotFrom, depositeSlotTo )

Проверяет, изменит ли предметы попытка их перемещения в хранилище. Актуально прежде всего для предметов с ограниченным сроком существования.

-- библиотека:
depositeBox

-- объявление:
function IsMoveModified( depositeSlotFrom, depositeSlotTo )

-- параметры:
depositeSlotFrom: number( int ) - индекс исходного слота с предметами в хранилище [0..]
depositeSlotTo: number( int ) - индекс целевого слота в хранилище [0..]

-- возвращаемые значения:
boolean - изменятся ли предметы

-- пример:
if depositeBox.IsMoveModified( 0, 1 ) then
  -- вывести предупреждение о возможном изменении предметов
end
