order.GetPreviousAchievementTop()

Возвращает описание лидера прошлого сезона по классу.

Для запроса данных с сервера необходимо использовать:

  • order.RequestPrestigeAchivements()

  • order.RequestRatingAchivements()

-- библиотека:
order

-- объявление:
function GetPreviousAchievementTop( ObjectId )

-- параметры:
ObjectId - идентификатор события

-- возвращаемые значения:
table or nil
  name : wstring - имя аватара
  achievement : number - количество очков
