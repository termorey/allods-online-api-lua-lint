group.CanKickMember()

Может ли главный игрок исключать игрока из группы. Возвращает true для лидера, при возможности голосования и т.п..

-- библиотека:
group

-- объявление:
function CanKickMember()

-- параметры:
нет

-- возвращаемое значение
boolean - true, если главный игрок может выгонять из группы

-- пример:
if group.CanKickMember() then
  group.KickMember( memberName )
end
