На главную

checkroomLib.GetItemCategories( itemId )

Возвращает категорию и коллекцию гардероба, в которой находится данный предмет. Возвращает данные для любого гардеробного предмета, не обязательно физически находящегося в гардеробе.

-- библиотека:
checkroomLib

-- объявление:
GetItemCategories( itemId )

-- параметры:
itemId: ObjectId - идентификатор предмета

-- возвращаемые значения:
table or nil - если предмет действительно гардеробные, то таблица с полями:
  categoryId: CheckRoomCategoryId - идентификатор ресурса категории
  collectionId: CheckRoomCollectionId - идентификатор ресурса коллекции

Пример:

local itemCategories = checkroomLib.GetItemCategories( itemId )
if itemCategories then
  LogInfo( itemCategories.categoryId:GetInfo().name )
end
