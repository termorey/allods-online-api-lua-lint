DNDConfirmDropAttempt()

Функция должна вызываться скриптом в ответ на событие EVENT_DND_DROP_ATTEMPT для подтверждения окончания drag&drop.

Замечание: Настоятельно рекомендуется вызывать данную функцию именно в том виджете, DnD которого активно в данный момент. В текущей реализации функция может быть вызвана из любого виджета с одинаковым эффектом, однако такое поведение не гарантируется в будущем.

-- класс
метод доступен для всех типов виджетов

-- семантика
function DNDConfirmDropAttempt()

-- параметры:
нет

-- возвращаемые значения:
нет

-- примеры:
wtPanel:DNDConfirmDropAttempt()
