EVENT_AVATAR_PROGRESS_GAINED

Приходит, если главный игрок получил один из типов опыта для развития. Это может быть обычный опыт (experience), репутация с какой-либо фракцией, слава за PvP сражения.

См. ENUM_DevelopmentTrack_..., ENUM_DevelopmentSource_....

-- поля:
type: number (enum ENUM_DevelopmentTrack_...) - тип опыта
sysType: string (enum "ENUM_DevelopmentTrack_...") - строковое представление типа опыта
source: number (enum ENUM_DevelopmentSource_...) - тип источника опыта
sysSource: string (enum "ENUM_DevelopmentSource_...") - строковое представление типа источника опыта
total: number - полученное количество опыта
bonus: number - сколько из этого опыта было получено за бонусные очки
level: number - уровень
value: number - текущее значение
isPvPExperience: boolean - этот опыт был получен за PvP

Также в некоторых случаях могут иметься дополнительные поля:
  Если изменилась репутация с для ENUM_DevelopmentTrack_Reputation:
    factionId: FactionId - фракция
