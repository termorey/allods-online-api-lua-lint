bulletinBoard.GetSectionInfos()

Возвращает информацию о доступных разделах доски объявлений.

-- библиотека:
bulletinBoard

-- объявление:
function GetSectionInfos()

-- параметры:
нет

-- возвращаемые значения:
таблица, содержащая информацию о доступных разделах, индексирована [0..], значения - таблицы со следующими полями:
 postTypeId: PostTypeId - идентификатор раздела доски объявлений
 sysName: string - служебное название раздела (для уникальной обработки)
 name: WString - локализованное название раздела
 description: WString - локализованное описание раздела
 lengthLimit: number (int) - максимальный размер одного объявления в символах

-- примеры:
local sectionInfos = bulletinBoard.GetSectionInfos()
for i = 0, GetTableSize( sectionInfos ) - 1 do
 local sectionInfo = sectionInfos[i]
 LogInfo( "раздел: ", sectionInfo.name )
end
