PlayRotationEffectSequence( self, sequenceParams )

Запускает цепочку эффектов поворота текстур виджета. Вращаются текстуры BackLayer и FrontLayer относительно центра виджета против часовой стрелки. Если текстуры выходят за пределы вписанной в виджет окружности, то для предотвращения обрезания текстур виджет должен иметь флаг clipContent = false.

Имеет тип ET_TEXTURE_ROTATION, соответствует шаблону PlayEffectSequence( self, sequenceParams )

Типы start / finish: number (float) - угол поворота в радианах

-- Класс
WidgetSafe

-- Типы шаблонных аргументов
start: number (float) or nil - начальный угол поворота виджета (в радианах)
finish: number (float) or nil - конечный угол поворота виджета (в радианах)

-- Пример
-- Плавно повернуть виджет на 180 градусов против часовой стрелки за 2 секунды, затем обратно по часовой стрелке за то же время
wt:PlayRotationEffectSequence{
  { 0, math.pi, 2000 },
  { nil, 0, 0, { type = EA_ACCELERATED, from = -0.5 * math.pi, to = -0.5 * math.pi } }
}
