На главную

customizationLib.IsActive()

Во время кастомизации какого-либо помещения или объекта возвращает true.

-- библиотека:
customizationLib

-- объявление:
function IsActive()

-- параметры:
нет

-- возвращаемые значения:
boolean - true, если находимся в состоянии кастомизации

-- пример:
if customizationLib.IsActive() then
  customizationLib.UninstallItem()
end
