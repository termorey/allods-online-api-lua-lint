family.GetSpellInfo( allodId )

Возвращает информацию о семейном заклинании.

-- библиотека:
family

-- объявление:
function GetSpellInfo( spellId )

-- параметры:
spellId: SpellId - идентификатор ресурса заклинания

-- возвращаемое значение
table or nil - если данное заклинание действительно семейное, то таблица с полями:
  experience: number (integer) - накопленный опыт
  level: number (integer) - уровень заклинания (зависит от накопленного опыта)
  levelExperience: number (integer) - опыт, необходимый для получения текущего уровня
  nextLevelExperience: number (integer) - опыт, необходимый для получения следующего уровня

-- пример:
local familySpell = family.GetSpellInfo( spellId )
if familySpell then
  local exp = familySpell.experience
end
