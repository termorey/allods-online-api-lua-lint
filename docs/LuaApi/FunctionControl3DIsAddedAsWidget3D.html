IsAddedAsWidget3D( self, widget )

Проверяет добавлен ли виджет как Widget3D в данный Control3DSafe.

-- класс
Control3DSafe

-- семантика
function IsAddedAsWidget3D( self, widget )

-- входные параметры:
self: Control3DSafe - виджет содержащий 3D сцену
widget: WidgetSafe - проверяемый виджет

-- возвращаемое значение:
bool - true если данный виджет добавлен как Widget3D к этому Control3DSafe

-- пример
if wtControl3D:IsAddedAsWidget3D( widget )
  wtControl3D:RemoveWidget3D( widget )
end
