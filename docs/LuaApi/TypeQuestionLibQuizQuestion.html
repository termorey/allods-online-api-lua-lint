QuizQuestion

Формат поля questionData у questionLib.GetInfo( questionId ) при значении поля type равном "!QuizQuestion":

table - таблица с полями:
  text: WString - локализованная строка, текст вопроса на который предполагается ответ игрока "да/нет"
  image: TextureId или nil - идентификатор текстуры для иконки, может отсутствовать
  type: String(ENUM_QuizQuestionType) - тип вопроса

Формат параметра result у questionLib.SendData( questionId, result ):

table - таблица с полями:
  choice: number (integer) - 1 если игрок ответил "да", 0 если игрок ответил "нет"
