EditLine. Описание строки ввода.

Строка ввода

Это элемент интерфейса предназначенный для ввода текстовой информации, наследованный от типа WidgetEditBase. Представляет собой редактируемую строку текстовой информации. Строка содержит курсор (мигание сделано переключением текстурных слоев) указывающий на то, где будет происходить та или иная операция с текстом. Помимо этого текст может быть полностью или частично выделен для операции над выделенной областью. Выделить текст можно либо мышью, либо зажав клавишу Shift и переместив курсор в строке.

Курсор перемещается внутри строки клавишами стрелок "вправо" и "влево". Клавиши "Home" и "End" позволяют перенести курсор в начало и конец строки, соответственно. Используйте Del или Backspace для удаления. Информация задаётся либо вводом текста с клавиатуры, либо копируется посредством выделения текста и комбинаций клавиш Ctrl+V для вставки, Ctrl+C для копирования, Shift+Del для вырезания, Tab для перехода к следующему в списке обхода контролу.

Дополнительные поля строки ввода:

  • isPassword: boolean - если true, то в строке ввода находится секретная информация и текст выводится в виде звездочек. По умолчанию false

Существующие реакции:

  • ReactionEnter: string - нажатие Enter

  • reactionUp: string - отпущена клавиша на клавиатуре

  • reactionDown: string - нажата клавиша на клавиатуре
