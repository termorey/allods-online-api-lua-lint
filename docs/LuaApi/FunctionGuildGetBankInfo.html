Пользовательские дополнения

На главную

guild.GetBankInfo()

Возвращает описание свойств гильдейского банка.

-- библиотека:
guild

-- объявление:
function GetBankInfo()

-- параметры:
нет

-- возвращаемое значение
table or nil - таблица с полями:
  realMinimumTransaction: number (integer) - минимальное реальное ограничение в данный момент по величине транзакции в меди
  minimumTransaction: number (integer) - ограничение на минимальное реальное ограничение по величине транзакции в меди
  maximumTransaction: number (integer) - максимальное ограничение по величине транзакции в меди
  maximumGuildMoney: number (integer) - максимальное количество меди в банке
  operationHistorySize: number (integer) - глубина истории операций
  moneyHistorySize: number (integer) - глубина истории оплат
  memberHistorySize: number (integer) - глубина истории по отдельному члену
  guildHistoryPageSize: number (integer) - максимальное количество записей на одной странице истории операций в банке
  guildHistorySize: number (integer) - текущее общее количество записей истории операций в банке

-- пример:
local realMinimumTransaction = guild.GetBankInfo().realMinimumTransaction
