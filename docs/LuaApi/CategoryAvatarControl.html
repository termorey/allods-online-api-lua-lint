CategoryAvatarControl

События и функции Lua API, относящиеся к управлению персонажем игрока.

Предусматривается два основных режима управления персонажем игрока (см. InterfaceMoveByClick).

Переключение режима управления предполагается осуществлять из интерфейса опций, глобальная переменная:

"use_move_by_click".
