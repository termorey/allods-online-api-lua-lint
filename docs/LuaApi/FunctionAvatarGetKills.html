Пользовательские дополнения

На главную

avatar.GetKills()

Возвращает информацию о количестве убитых в PvP противников. При изменении приходит EVENT_VICTIM_TRACKER_CHANGED.

-- библиотека:
avatar

-- объявление:
function GetKills()

-- параметры:
нет

-- возвращаемые значения
number (integer) или nil - nil если нет информации; иначе число PvP-убийств, совершённых аватаром за всю карьеру


-- пример:
local kills = avatar.GetKills()
