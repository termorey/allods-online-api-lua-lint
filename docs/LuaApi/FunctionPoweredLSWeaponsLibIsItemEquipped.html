poweredLSWeaponsLib.IsItemEquipped( itemId )

Возвращает true, если этот или такой-же предмет надет на персонажа.

-- библиотека:
poweredLSWeaponsLib

-- объявление:
IsItemEquipped( itemId )

-- параметры:
itemId: ObjectId - идентификатор предмета

-- возвращаемые значения:
boolean - true, если предмет надет

Пример:

local isItemEquipped = poweredLSWeaponsLib.IsItemEquipped( itemId )
