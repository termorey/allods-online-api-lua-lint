avatar.GetExperience()

Возвращает информацию об опыте персонажа

-- библиотека:
avatar

-- объявление:
function GetExperience()

-- параметры:
нет

-- возвращаемые значения:
table, поля:
currentExp: number (int) - текущий опыт персонажа
currentLevelExp: number (int) - опыт для получения текущего уровня персонажа
nextLevelExp: number (int) - опыт для получения следующего уровня персонажа
extraExp: number (int) - накопленный бонусный опыт, который может быть переведен в текущий(currentExp) при помощи "Фолианта знаний"
extraExpCap: number (int) - максимально возможное значение накопленного бонусного опыта(extraExp)
bonus: number (float) - множитель получаемого опыта (1.0 - опыт по умолчанию)
pvpExp: number (int) - опыт полученный за убийства других игроков

-- пример:
local theXp = avatar.GetExperience()
