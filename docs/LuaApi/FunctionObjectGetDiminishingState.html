object.GetDiminishingState( objectId )

Возвращает информацию о диминишинге(сопротивлении контролю) объекта.

-- библиотека
object

-- объявление
function GetDiminishingState( objectId )

-- параметры
objectId: ObjectId - идентификатор объекта

-- возвращаемые значения
table of Number - таблица в которой ключом является группа ENUM_DiminishingGroup, а значением уровень диминишинга(от 0 до 3)

-- пример
local diminishingState = object.GetDiminishingState( avatar.GetId() )
