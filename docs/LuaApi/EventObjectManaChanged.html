EVENT_OBJECT_MANA_CHANGED

Событие присылается при изменении процентного значения маны интерактивных объектов и числового значения маны для персонажа игрока

-- поля:
id: ObjectId (not nil) - идентификатор объекта
manaDelta: number (integer) or nil - на сколько изменилась точная величина маны, если такая информация доступна для этого объекта
manaLimitDelta: number (integer) or nil - на сколько изменилась максимальная величина маны, если такая информация доступна для этого объекта
manaPercentsDelta: number (integer) or nil - на сколько изменилась относительная величина маны, если такая информация доступна для этого объекта

См. также:

  • object.GetManaInfo( objectId )

  • EVENT_OBJECT_HEALTH_CHANGED
