remortCreation.IsZoomEnabled()

Возвращает true, если можно назумиться на лицо при выборе или создании персонажа. Также в таком состоянии можно назумливаться колесом мыши.

При изменении состояния приходит EVENT_REMORT_ZOOM_TRIGGER_CHANGED.

-- библиотека:
remortCreation

-- объявление:
function IsZoomEnabled()

-- параметры:
нет

-- возвращаемые значения:
boolean - true, если можно назумиться на лицо

-- примеры:
if remortCreation.IsZoomEnabled() and not remortCreation.IsZoomOnFace() then
  remortCreation.ToggleZoom()
end
