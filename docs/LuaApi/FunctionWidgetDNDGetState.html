DNDGetState()

Возвращает состояние виджета в системе drag&drop. Описание всех возможных состояний смотреть в DND_STATE_...

-- класс
метод доступен для всех типов виджетов

-- семантика
function DNDGetState( self )

-- параметры:
нет

-- возвращаемые значения:
number (enum DND_STATE_...) - состояние виджета в системе drag&drop

-- примеры:
if wtPanel:GetDNDState() ~= DND_STATE_NOT_REGISTERED then
  wtPanel:DNDUnregister()
end
