Пользовательские дополнения

На главную

questionLib.GetQuestions()

Выдаёт список вопросов, на которые в данный момент ожидается ответ от игрока.

-- библиотека:
questionLib

-- объявление:
GetQuestions()

-- параметры:
нет

-- возвращаемые значения:
table - список (индексируется начиная с 0) ObjectId, соответствующих вопросам


-- пример:
local questions = questionLib.GetQuestions()
if questions[0] then
  local question = questionLib.GetInfo( questions[0] )
end
