EVENT_FATALITY_PHRASE_RESULT

Событие присылается в ответ на выполнение функции avatar.SetFatalityPhrase

-- поля:
failCause: string - ActionFailCause результата выполнения функции
result: string - причина ошибки
substring: wstring - часть строки, в которой была ошибка
