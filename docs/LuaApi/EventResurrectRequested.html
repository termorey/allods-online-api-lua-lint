EVENT_RESURRECT_REQUESTED

Этот запрос посылается, когда кто-то пытается оживить главного игрока. В ответ игрок должен ответить avatar.ResurrectReply( accept ). Если игрок не успеет, то прийдёт EVENT_RESURRECT_REJECTED.

-- поля:
resurrecterName: WString - имя юнита, пытающегося оживить игрока.
unitId: ObjectId (not nil) - Id юнита, пытающегося оживить игрока.
timeOutMs: number (integer) - сколько миллисекунд действует это приглашение

Search: "CategoryLuaApi" "CategoryClassSpecific"
