InsertTextAtCursorPos( self, text )

Вставляет текст в текущей позиции курсора EditLine.

Если в EditLine был выделен фрагмент текста, то этот фрагмент будет заменён вставляемым текстом.

-- класс
EditLineSafe

-- семантика
function InsertTextAtCursorPos( self, text )

-- входные параметры:
text: WString -- вставляемый текст

-- возвращаемое значение:
отсутствует

-- пример
wtEditLine:InsertTextAtCursorPos( debugCommon.ToWString( "текст" ) )
