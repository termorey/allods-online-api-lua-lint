Пользовательские дополнения

На главную

runeLib.GetSlotInfo( dressSlot )

Возвращает таблицу с информацией о слоте одежды под руну.

См. также: DRESS_SLOT_...

-- библиотека:
runeLib

-- объявление:
function GetSlotInfo( dressSlot )

-- параметры:
dressSlot: number (enum DRESS_SLOT_...) - номер слота

-- возвращаемые значения:
table or nil - таблица с информацией о слоте под руну или nil в случае ошибок
  dressSlot: number (enum DRESS_SLOT_...) - номер слота
  requiredLevel: number (integer) - уровень игрока, с которого руна в этом слоте начнёт работать
  isOffensive: boolean - этот слот для атакующей руны

-- пример:
local runeSlotInfo = runeLib.GetSlotInfo( dressSlot )
if runeSlotInfo then
  local requiredLevel = runeSlotInfo.requiredLevel
end
