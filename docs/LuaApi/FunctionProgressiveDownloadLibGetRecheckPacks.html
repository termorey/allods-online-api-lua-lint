progressiveDownloadLib.GetRecheckPacks()

Возвращает список паков, отправленных на перепроверку/починку/перекачивание.

-- библиотека:
GetRecheckPacks

-- объявление:
GetRecheckPacks()

-- параметры:
нет

-- возвращаемые значения:
набор целочисленных идентификаторов паков

-- пример:
local packs = progressiveDownloadLib.GetRecheckPacks()
; local zeroPackId = packs[0]
; local zeroPackInfo = GetPackInfoById( zeroPackId )
