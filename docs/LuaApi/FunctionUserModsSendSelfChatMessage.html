Пользовательские дополнения

На главную

userMods.SendSelfChatMessage( message, messageLevel )

Выводит сообщение в чат в каналы группы Системные -> Дополнения. Сообщение не пересылается другим игрокам.

Функцию можно использовать только в пользовательских дополнениях.

-- библиотека:
userMods

-- объявление:
function SendSelfChatMessage( message, messageLevel )

-- параметры:
message: WString - текст сообщения
messageLevel: string or nil - имя канала, по умолчанию message

-- возвращаемые значения:
Нет

-- примеры:
userMods.SendSelfChatMessage( addonErrorMessage, "error" )

На данный момент поддержаны каналы

message - Сообщения (LogColorWhite)
notice - Уведомления (LogColorYellow)
warning - Предупреждения (LogColorOrange)
error - Ошибки (LogColorRed)
