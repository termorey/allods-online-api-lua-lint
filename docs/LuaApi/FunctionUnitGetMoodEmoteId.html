Пользовательские дополнения

На главную

unit.GetMoodEmoteId( unitId )

Если у указанного игрока выставлено какое-либо настроение, то метод возвращает идентификатор эмоции, с помощью которой выставлено это настроение.

-- библиотека
unit

-- объявление:
function GetMoodEmoteId( unitId )

-- параметры
unitId: ObjectId - идентификатор игрока, для которого надо получить эмоцию настроения

-- возвращаемые значения
ObjectId or nil - идентификатор эмоции, если есть какое-либо настроение у игрока

-- пример использования
local emoteId = unit.GetMoodEmoteId( unitId )
local emoteInfo = avatar.GetEmoteInfo( emoteId )
if emoteInfo then
  common.LogInfo( "Emote system name: " .. emoteInfo.sysName )
end
