Особенности Lua runtime

В качестве Lua runtime используется '''luajit-2.0.5'''.

Особенности и отличия от стандартной конфигурации

  • Список доступных встроенных библиотек: base, math, string, table, bit, jit

  • Нестандартные флаги сборки: LUAJIT_ENABLE_LUA52COMPAT, LUAJIT_DISABLE_FFI, LUAJIT_DISABLE_JITUTIL, LUAJIT_DISABLE_VMEVENT

  • Запрещено неявное объявление глобальных переменных, для объявление глобальных переменных надлежит использовать Global( name, value )

  • Обращение к несуществующим глобальным переменным порождает ошибку

  • Ряд функций встроенных библиотек недоступен: getfenv, setfenv, load, loadfile, dofile, newproxy, coroutine.*, jit.attach, jit.util.*

  • Во встроенные библиотеки добавлен ряд функция: math.sign( number ), math.round( nubmer, [limit] ), math.clamp( number, [min], [max] ), math.wrap( number, min, max ), table.clone( tbl )
