ET_...

Типы эффектов для виджетов. Определены в коде.

ET_MOVE - эффект изменения положения (WidgetPlacement: posX, posY, highPosX, highPosY) виджета
ET_FADE - эффект изменения прозрачности виджета/текста
ET_RESIZE - эффект изменения размеров (WidgetPlacement: sizeX, sizeY) виджета
ET_TEXTURE_ROTATION - эффект изменения угла поворота текстур (BackLayer, FrontLayer) виджета
ET_TEXT_SCALE - эффект изменения размера текста, только для TextView
