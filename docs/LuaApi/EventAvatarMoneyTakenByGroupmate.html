EVENT_AVATAR_MONEY_TAKEN_BY_GROUPMATE

Сообщение приходит, когда согрупник или соотрядник главного игрока получает деньги.

Внимание! сообщение приходит только в UIState.

-- система:
UIState

-- поля:
playerId: ObjectId or nil - идентификатор согрупника, если возможно его получить
money: number (integer) - количество денег
actionType: string (enum "ENUM_EnumTakeItemActionType_...") - откуда взялся предмет

См. "ENUM_EnumTakeItemActionType_...".
