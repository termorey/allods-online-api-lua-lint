На главную

options.SetOptionCurrentIndex( optionId, index )

Задает новое значение для опции. Новое значение будет использовано при выполнении команды Apply.

-- библиотека:
options

-- объявление:
function SetOptionCurrentIndex( optionId, index )

-- параметры:
optionId: ObjectId - идентификатор опции
index: number (int) - новое значение для опции - для дискретного типа данных

-- возвращаемые значения:
нет

-- пример:
options.SetOptionCurrentIndex( optionId, index )
