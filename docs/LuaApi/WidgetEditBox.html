Многострочное поле ввода

Это элемент интерфейса предназначенный для ввода текстовой информации, наследованный от типа WidgetEditBase. Представляет собой редактируемое многострочное поле для ввода текстовой информации. Поле содержит курсор (мигание сделано переключением текстурных слоев) указывающий на то, где будет происходить та или иная операция с текстом. Помимо этого текст может быть полностью или частично выделен для операции над выделенной областью. Выделить текст можно либо мышью, либо зажав клавишу Shift и переместив курсор в строке, а так же с помошью сочетания клавиш Ctrl+A.

Курсор перемещается внутри поля с текстом клавишами стрелок Up(▲), Down(▼), Left(◄), Right(►). Клавиши Home и End позволяют перенести курсор в начало и конец строки, а комбинации Ctrl+Home и Ctrl+End на начало и конец текста, соответственно. Используйте Del или Backspace для удаления. Информация задаётся либо вводом текста с клавиатуры, либо копируется посредством выделения текста и комбинаций клавиш Ctrl+V для вставки, Ctrl+C для копирования, Ctrl+X для вырезания, Tab для перехода к следующему в списке обхода контролу. Клавиша Enter служит для переноса ввода на новую строку.

Дополнительные поля виждета:

  • scrollBar: WidgetGlideScrollBar - плавный скроллбар для скролирования теста по вертикали (не обязательный), отнимает часть внутреннего пространства виджета
