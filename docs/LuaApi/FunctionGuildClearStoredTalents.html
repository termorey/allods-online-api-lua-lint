guild.ClearStoredTalents( < field, row, column > )

Сбрасывание выбранных на изучение вех.

-- библиотека:
guild

-- объявление:
function ClearStoredTalents( [ field, row, column ] )

-- параметры:
нет - если нужно сбросить все вехи на изучение
field: number (int) - номер поля
row: number (int) - номер строки
column: number (int) - номер столбца

-- возвращаемые значения
нет

-- пример
guild.ClearStoredTalents() -- сбросить все выбранные на изучение вехи
guild.ClearStoredTalents( 0, 4, 5 ) -- сбросить веху 4:5 в поле 0 и все сопутствующие
