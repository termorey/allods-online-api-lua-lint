Пользовательские дополнения

На главную

cartographer.CanNavigateToPoint( zonesMapId )

Можно ли найти путь для этой карты.

-- библиотека:
cartographer

-- объявление:
function CanNavigateToPoint( zonesMapId )

-- параметры:
zonesMapId: ObjectId - идентификатор интерфейсной карты

-- возвращаемые значения
boolean - можно найти путь к точке
--}

Пример:

if cartographer.CanNavigateToPoint( questLocation.zonesMapId ) then
 cartographer.NavigateToPoint( questLocation.zonesMapId, questLocation.position )
end
