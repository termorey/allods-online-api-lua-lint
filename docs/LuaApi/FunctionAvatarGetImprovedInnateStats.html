avatar.GetImprovedInnateStats()

Аналогично avatar.GetInnateStats() дает основные характеристики аватара, но с учётом намеченных к распределению (ещё нераспределённых окончательно) пунктов характеристик. Работает только когда avatar.CanImproveInnateStats() дает true.

-- библиотека:
avatar

-- объявление:
GetImprovedInnateStats()

-- параметры:
нет

-- возвращаемые значения:
table или nil - если нет информации, то nil; иначе таблица, аналогичная той, что даёт avatar.GetInnateStats()


-- пример:
local stats = avatar.GetImprovedInnateStats()

См. также:

  • avatar.GetInnateStats()

  • avatar.CanImproveInnateStats()

  • avatar.ImproveInnateStat( stat, amount )
