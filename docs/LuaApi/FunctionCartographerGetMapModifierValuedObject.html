cartographer.GetMapModifierValuedObject( mapModifierId )

Возвращает новый экземпляр ValuedObject для модификатора карты.

Тип объекта - VAL_OBJ_TYPE_MAP_MODIFIER.
Текст - имя модификатора карты.
Изображение - иконка модификатора карты.
Id: MapModifierId - Id ресурса модификатора карты.

Замечание: поскольку ValuedObject это userdata, то Lua работает с ними по ссылке, а не по значению.

-- библиотека:
cartographer

-- объявление:
function GetMapModifierValuedObject( mapModifierId )

-- параметры:
mapModifierId: MapModifierId - Id модификатора карты

-- возвращаемые значения:
ValuedObject - экземпляр ValuedObject

-- примеры:
local mapModifiers = cartographer.GetCurrentMapModifiers()
local valuedObject = mapModifiers[0] and cartographer.GetMapModifierValuedObject( mapModifiers[0] )

См. также MapModifierId, cartographer.GetCurrentMapModifiers() и EVENT_UNIT_DAMAGE_RECEIVED
