EVENT_PROGRESSIVEDOWNLOADING_TOPPACKS_CHANGED

Сообщает об изменении списка наиболее приоритетных для скачивания паков, либо параметров одного из них.

-- поля:
нет

См. GetTopRequestedPacks() - получить часть списока наиболее приоритетных паков для скачивания ("окно" должно быть задано заранее). Возвращается набор целочисленных идентификаторов паков.

SetTopPacksWindow( start, size ) - задать "окно" для мониторинга изменений в списке наиболее приоритетных паков на скачивание

GetPackInfoById( id ) - получить инфу о паке по его целочисленному идентификатору
