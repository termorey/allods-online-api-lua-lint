Пользовательские дополнения

На главную

tutorialLib.GetTutorialContent( id )

Возвращает содержимое обучения и данные необходимые для его отображения.

-- библиотека
tutorialLib

-- объявление:
function GetTutorialContent( id )

-- параметры
id: TutorialId(ResourceId) - идентификатор обучения

-- возвращаемые значения
name: WString or nil - отображаемое название обучения
text: ValuedText or nil - полный текст обучения
shortText: ValuedText or nil - короткий текст обучения
image: TextureId or nil - иллюстрация обучения
link: ValuedText or nil - внешняя ссылка на дополнительную информацию по обучению
bind: table or nil - если бинд задан в ресурсе то таблица с полями:
 section: String - имя секции
 name: String - имя бинда
popUps: table of tables or nil - массив данных для всплывающих сообщений, индексирован с 1, каждый элемент массива содержит поля:
  text: ValuedText - короткий текст всплывающего сообщения
  target: String - системное имя целевого виджета для всплывающего сообщения

-- пример использования
local testTutorialContent = tutorialLib.GetTutorialContent( tutorialLib.GetTutorialBySysName( "test" ) )
