На главную

avatar.UseItemOnMountAndTakeActions( itemId, mountId, useAction )

Использовать выбранную вещь на маунта и выполнить какие-то воздействия.

-- библиотека:
avatar

-- объявление:
function UseItemOnMountAndTakeActions( itemId, mountId, useAction )

-- параметры:
itemId: ObjectId - идентификатор предмета, который применяется на маунта
mountId: ObjectId - идентификатор маунта, на который будет применятся предмет

-- необязательные параметры:
useAction: number - индекс (sysIndex) выбранного Usage (по умолчанию - -1)

-- возвращаемые значения:
нет

-- пример:
avatar.UseItemOnMountAndTakeActions( itemId, mountId, 3 )
