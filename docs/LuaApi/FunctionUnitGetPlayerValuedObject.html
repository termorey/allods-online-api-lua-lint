unit.GetPlayerValuedObject( name, shardName, uniqueId )

Возвращает новый экземпляр ValuedObject по имени игрока.

Тип объекта - VAL_OBJ_TYPE_PLAYER.
Текст - имя игрока.
Изображение - нет.
Id: WString - имя игрока.

Замечание: поскольку ValuedObject это userdata, то Lua работает с ними по ссылке, а не по значению.

-- библиотека:
unit

-- объявление:
function GetPlayerValuedObject( name, shardName, uniqueId )

-- параметры:
name: WString - имя персонажа
shardName: WString - имя шарда персонажа(необязательный параметр)
uniqueId: WString - уникальный идентификатор персонажа(необязательный параметр)

-- возвращаемые значения:
ValuedObject - экземпляр ValuedObject

shardName - необязательный параметр, который можно завернуть в ValuedObject и доставать с помощью GetShardName.

uniqueId - аналогично извлекается с помощью GetId.

-- примеры:
local valuedObject = unit.GetPlayerValuedObject( playerName, playerShardName )
