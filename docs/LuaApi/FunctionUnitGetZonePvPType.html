unit.GetZonePvPType(id)

Возвращает информацию о PvP-типе зоны для персонажа.

-- библиотека:
unit

-- объявление:
function GetZonePvPType( unitId )

-- параметры:
unitId: ObjectId (not nil) -- идентификатор юнита

-- возвращаемые значения
number (enum ZONE_PVP_TYPE_...) - PvP состояние


-- пример:
local pvp = unit.GetZonePvPType( unitId )

Имеющиеся типы зон:

ZONE_TYPE_ASTRAL
ZONE_TYPE_ARENA
ZONE_TYPE_CONTESTED
ZONE_TYPE_FREE_PVP
ZONE_TYPE_GRAVEYARD
ZONE_TYPE_GUILD_BATTLEGROUND
ZONE_TYPE_HOMELAND
ZONE_TYPE_SANCTUARY
