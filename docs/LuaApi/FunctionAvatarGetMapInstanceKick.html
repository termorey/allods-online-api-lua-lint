Пользовательские дополнения

На главную

avatar.GetMapInstanceKick()

Возвращает информацию о причине и времени до выброса главного игрока из инстанса. Если счетчика нет, возвращает nil.

См. также "ENUM_AvatarKickCause...".

-- библиотека:
avatar

-- объявление:
function GetMapInstanceKick()

-- параметры:
нет

-- возвращаемые значения:
table or nil – информация о счётчике или nil, если счетчика нет. Таблица с полями:
  durationMs: number (integer) - общее время на выброс из инстанса в миллисекундах
  remainingMs: number (integer) - сколько времени осталось до выброса в миллисекундах
  sysCause: string ("ENUM_AvatarKickCause...") - причина выброса

-- пример:
local timeInfo = avatar.GetMapInstanceKick()
if timeInfo then
 LogInfo( "Оставшееся время до выброса из инстанса: ", timeInfo.remainingMs / 1000 )
end
