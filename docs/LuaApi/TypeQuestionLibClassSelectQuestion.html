ClassSelectQuestion

Вопрос для выбора класса персонажа (например в процессе смены класса). Списка классов, из которых нужно выбирать, в этих данных не имеется. Предполагается, что он будет получен другими методами Lua API, например remortCreation.GetAvatarTemplatesInfo().

Формат поля questionData у questionLib.GetInfo( questionId ) при значении поля type равном "!ClassSelectQuestion":

table - таблица с полями:
  text: WString - локализованная строка, текст, информирующий игрока о смысле требуемого от него выбора

Формат параметра result, который нужно передать в questionLib.SendData( questionId, result ) в ответ на вопрос этого типа:

table - таблица с полями:
  characterClass: CharacterClassId (ResourceId) - идентификатор класса

См. также

  • remortCreation.GetAvatarTemplatesInfo() - поля factions[0..].races[0..].classes[0..].classId
