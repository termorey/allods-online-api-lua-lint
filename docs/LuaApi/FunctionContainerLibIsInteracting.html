containerLib.IsInteracting( slotType )

Возможно ли открытие контейнера с помощью NPC, т.е. находимся ли мы в состоянии разговора с соответствующим персонажем.

Список контейнеров см. в ITEM_CONT_....

-- библиотека:
containerLib

-- объявление:
function IsInteracting( slotType )

-- параметры:
slotType: number (enum ITEM_CONT_...) - тип контейнера с предметами

-- возвращаемые значения:
boolean - находимся ли в состоянии разговора с подходящим персонажем (позволяющим открыть контейнер)

-- пример:
if containerLib.IsInteracting( ITEM_CONT_DEPOSITE ) and not containerLib.IsOpen( ITEM_CONT_DEPOSITE ) then
 containerLib.OpenByInteractor( ITEM_CONT_DEPOSITE )
end
