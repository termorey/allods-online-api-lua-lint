avatar.SetPetMoveMode( mode )

Установливает для пета аватара тип движения:

PET_MOVE_FOLLOW - следовать за хозяином,
PET_MOVE_STAY - оставаться на месте.


-- библиотека
avatar

-- объявление:
function SetPetMoveMode( mode )

-- параметры
mode: number ( enum PET_MOVE_XXX ) - тип движения

-- возвращаемые значения
нет

-- пример использования
avatar.SetPetMoveMode( PET_MOVE_FOLLOW )
