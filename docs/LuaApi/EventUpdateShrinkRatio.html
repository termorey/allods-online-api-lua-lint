EVENT_UPDATE_SHRINK_RATIO

Событие об изменении размеров экрана и необходимости изменить коэффициент масштабирования пользовательского интерфейса - см. функцию SetShrinkRatio (закрытая ссылка).

DEPRECATED: Событие устарело и помечено к удалению. Масштабирование реализовано иным способом.

-- поля:
sizeX: number (int) - размер экрана в реальных координатах по X
sizeY: number (int) - размер экрана в реальных координатах по Y
