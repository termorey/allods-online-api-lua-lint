guild.CanUpdateTalent( field, row, column )

Можно ли проапдейтить (или выучить) талант гильдии из поля талантов.

-- библиотека:
guild

-- объявление:
function CanUpdateTalent( field, row, column )

-- параметры:
field: number (int) - номер поля
row: number (int) - номер строки
column: number (int) - номер столбца

-- возвращаемые значения
boolean - можно ли проапдейтить (или выучить) талант гильдии

-- пример
local canUpdate = guild.CanUpdateTalent( field, row, column )
