binding.StartRebind( sysSectionName, sysBindName, index )

Включает режим распознавания горячих клавиш. После удачного распознавания можно применить бинд к текущей конфигурации.

Ограничения. Сторонним аддонам запрещено использовать данную функцию пока персонаж игрока находится в бою.

-- библиотека:
binding

-- объявление:
function StartRebind( sysSectionName, sysBindName, index )

-- параметры:
sysSectionName: string - идентификатор секции
sysBindName: string - идентификатор бинда (реакции)
index: number (int) - индекс горячей клавиши

-- возвращаемые значения:
нет

-- пример:
binding.StartRebind( sysSectionName, sysBindName, 1 )
