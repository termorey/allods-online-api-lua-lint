mount.GetActiveTimeLeftMs()

Возвращает время в миллисекундах до исчезновения вызванного ездового животного главного игрока из конюшни.

-- библиотека:
mount

-- объявление:
function GetActiveTimeLeftMs()

-- параметры:
нет

-- возвращаемые значения:
number (integer) or nil - время в миллисекундах до исчезновения животного из конюшни, если у него есть срок жизни

-- пример:
local timeLeftMs = mount.GetActiveTimeLeftMs()
if timeLeftMs then
  LogInfo( "Осталось ", timeLeftMs / 1000, " секунд" )
end
