Дискретный слайдер

WidgetDiscreteSlider - это дискретная реализация базового слайдера WidgetSlider.

Дополнительные поля:

  • stepsCount: number (integer) - количество дискретных позиций. По умолчанию 0

Существующие реакции:

Слайдер не имеет никаких дополнительных реакций.
