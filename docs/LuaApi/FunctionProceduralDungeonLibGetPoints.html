На главную

proceduralDungeonLib.GetPoints( goalId )

Возвращает список ключевых точек для выбранного задания процедурного данжа. Возвращает все точки если идентификатор задания( goalId ) не задан.

-- библиотека:
proceduralDungeonLib

-- объявление:
function GetPoints( goalId )

-- необязательные параметры:
goalId: ObjectId or nil - идентификатор задания

-- возвращаемое значение
table or nil - массив каждый элемент которого; индексируется начиная с 1
 GamePosition - координаты точки в игровом пространстве

-- пример:
local allPoints = proceduralDungeonLib.GetPoints()
