common.GetStateRelatedTextGroup( sysGroup, optional )

Возвращает группу связанных с состоянием текстовых ресурсов. Группа представляет собой userdata (RelatedTextsLua).

-- библиотека:
common

-- объявление:
function GetStateRelatedTextGroup( sysGroup, optional )

-- параметры:
sysGroup: string or nil - идентификатор группы
optional: boolean or nil - если true, то отсутствие искомой группы не считается ошибкой (по умолчанию false)

-- возвращаемые значения:
RelatedTextsLua or nil - группа текстов или nil в случае если группа не найдена

-- пример:
Смотреть в GetAddonRelatedTextGroup

См. описание группы текстовых ресурсов: RelatedTextsLua
