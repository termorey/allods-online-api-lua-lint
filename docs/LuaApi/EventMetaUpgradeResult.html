EVENT_META_UPGRADE_RESULT

Уведомление о результатах попытки улучшения мета-предмета.

-- поля:
sysCause: string (enum "ENUM_ActionFailCause_...") - результат попытки улучшения, в случае удачи приходит "ENUM_ActionFailCause_NoFail"
isLucky: boolean - сработал ли шанс моментального/двойного улучшения
improvementIncrease: number (float) - степень улучшения (в промежутке [0, 99])
upgradeChanceBehavior: string (enum ENUM_UpgradeChanceBehavior_...) - поведение при срабатывании шанса моментального/двойного улучшения
upgradeVector: string (enum "META_UPGRADE_...") - что улучшается: качество или уровень

В случае удачи приходят следующие поля:
 sourceId: ObjectId - идентификатор улучшаемого предмета
 enchancerId: ObjectId - идентификатор предмета-улучшителя
 agentId: ObjectId - идентификатор предмета реагента
 agentCount: number (integer) - количество использованных предметов-реагентов
 resultId: ObjectId - идентификатор полученного предмета

См. также:

  • avatar.UpgradeMetaItem.

  • avatar.GetMetaUpgradeResults( sourceId, enhancerId, agentId, agentCount ).

  • "ENUM_ActionFailCause_...".

  • ENUM_UpgradeChanceBehavior_....

  • "META_UPGRADE_...".
