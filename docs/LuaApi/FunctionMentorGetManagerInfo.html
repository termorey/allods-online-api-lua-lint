На главную

mentor.GetManagerInfo()

Возвращает информацию о активности главного игрока, связанной с менторством.

Идентификатор описателя субъекта менторских взаимоотношений определяет этого субъекта. Им может быть как ментор, так и ученик.

См. mentor.GetPlayerInfo( mentorPlayerId ).

-- библиотека:
mentor

-- объявление:
function GetManagerInfo()

-- параметры:
нет

-- возвращаемые значения -
table or nil - если не произошло ошибки, то таблица с полями:
  apprenticeMentorId: ObjectId  - идентификатор субъекта менторских взаимоотношений главного игрока, у которого главный игрок состоит учеником
  mentorId: ObjectId  - идентификатор субъекта менторских взаимоотношений, которым является сам главный игрок будучи ментором
  avatarApprenticeId: ObjectId  - идентификатор субъекта менторских взаимоотношений, которым является сам главный игрок будучи учеником какого-то ментора
  apprenticeFlag: boolean - true, если главный игрок выставил флаг, что он хочет быть учеником
  mentorFlag: boolean - true, если главный игрок выставил флаг, что он хочет быть ментором
  canBeMentor: boolean - true, если главный игрок может быть ментором
  canBeApprentice: boolean - true, если главный игрок может быть учеником

-- пример:
local info = mentor.GetManagerInfo()
if info.mentorId then
  LogInfo( "Я - Учитель" )
end
