На главную

unit.GetManaPercentage( unitId )

Возвращает информацию об мане персонажа в процентах. Если значение маны неизвестно, возвращается nil.

-- библиотека:
unit

-- объявление:
function GetManaPercentage( unitId )

-- параметры:
unitId: ObjectId (not nil) - идентификатор юнита

-- возвращаемые значения:
percents: number (int) or nil - процент текущей маны от максимальной

-- пример:
unit.GetManaPercentage( unitId )
