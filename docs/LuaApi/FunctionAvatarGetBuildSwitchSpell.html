avatar.GetBuildSwitchSpell()

Получить id спелла, переключающего специализацию (билд) аватара. Используется для получения информации об условиях переключения.

-- библиотека:
avatar

-- объявление:
function GetBuildSwitchSpell()

-- параметры:
нет

-- возвращаемые значения
QuestId - id спелла, переключающего специализацию аватара

-- пример
LogInfo( spellLib.GetRequirements(avatar.GetBuildSwitchSpell()) )
