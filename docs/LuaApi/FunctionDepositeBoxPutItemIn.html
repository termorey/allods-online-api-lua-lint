depositeBox.PutItemIn( itemId, depositeSlotTo, countNoMore )

Попытка поместить предметы в хранилище.

-- библиотека:
depositeBox

-- объявление:
function PutItemIn( itemId, depositeSlotTo, countNoMore )

-- параметры:
itemId: ObjectId - id предмета; предмет должен находиться в одном из контейнеров, например в сумке или переполненной сумке
depositeSlotTo: number( int ) or nil - индекс целевого слота в хранилище [0..] или nil, если слот не важен
countNoMore: number( int ) or nil - максимальное количество предметов из пачки для переноса или nil, если нужно положить всю пачку

-- возвращаемые значения:
нет

-- пример:
depositeBox.PutItemIn( itemId, nil, nil )
