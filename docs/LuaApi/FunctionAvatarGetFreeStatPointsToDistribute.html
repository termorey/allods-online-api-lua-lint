avatar.GetFreeStatPointsToDistribute()

Даёт количество доступных для распределения пунктов характеристик (ещё не намеченных к распределению). Работает только если avatar.CanImproveInnateStats() даёт true.

-- библиотека:
avatar

-- объявление:
GetFreeStatPointsToDistribute()

-- параметры:
нет

-- возвращаемые значения:
Number (integer) или nil - если нет информации, то nil; иначе количество доступных для распределения пунктов характеристик


-- пример:
local points = avatar.GetFreeStatPointsToDistribute()

См. также:

  • avatar.CanImproveInnateStats()
