WidgetButtonVariant. Вариант кнопки.

Кнопка содержит несколько вариантов кнопки, между которыми она может переключаться.

Поля описания одного варианта:

    • TextFileRef: TextFileRef - ссылка на текстовый файл с текстом кнопки

    • ReactionOnUp: boolean - если true, то реакцию о клике присылать при отпускании кнопки, иначи при нажатии. По умолчанию false

    • LayerHighlight: WidgetLayer - текстура подсветки

    • PushedOffset: vec2 - смещение при нажатии

    • StateNormal: WidgetButtonState - обычное состояние кнопки

    • StatePushed: WidgetButtonState - нажатое состояние кнопки

    • StateHighlighted: WidgetButtonState - подсвеченное, но не нажатое состояние кнопки

    • StatePushedHighlighted: WidgetButtonState - подсвеченное и нажатое состояние кнопки

    • StateDisabled: WidgetButtonState - отключенное (запрещено нажимать) состояние кнопки

    • soundOver: WidgetSoundBase - звук, воспроизводимый при наведении на кнопку

    • soundPress: WidgetSoundBase - звук, воспроизводимый при нажатии на кнопку

Реакции:

  • Reaction: string - реакция на клик LMB

  • ReactionRightClick: string - реакция на клик RMB

  • reactionDblClick: string - реакция на двойной клик
