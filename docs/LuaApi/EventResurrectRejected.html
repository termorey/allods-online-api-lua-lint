EVENT_RESURRECT_REJECTED

Этот запрос посылается, когда кто-то пытается оживить главного игрока, и игрок не успел ответить avatar.ResurrectReply( accept ).

-- поля:
resurrecterName: WString - имя юнита, пытавшегося оживить игрока.
unitId: ObjectId (or nil) - Id юнита, пытавшегося оживить игрока, если он есть в реплике.

Search: "CategoryLuaApi" "CategoryClassSpecific"
