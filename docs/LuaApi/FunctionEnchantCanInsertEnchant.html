На главную

enchant.CanInsertEnchant( enchantItemId, itemId )

Показывает, можно ли вставить данный энчант в предмет.

-- библиотека
enchant

-- объявление:
function CanInsertEnchant( enchantItemId, itemId )

-- параметры
enchantItemId: ObjectId - идентификатор энчанта
itemId: ObjectId - идентификатор проверяемого предмета

-- возвращаемые значения
table или nil - nil если нет информации; иначе таблица с полями:
  success: boolean - если true, то энчант можно вставить в предмет
  cause: string ("ENUM_EnchantInsertionResult_...") - причина, по которой нельзя вставить энчант

-- пример использования
local canDo = CanInsertEnchant( enchantItemId, itemId )
