avatar.IsAlchemyComponentsReady()

Возвращает true, если в алхимическом приборе достаточно реагентов для начала реакции.

-- библиотека
avatar

-- объявление:
function IsAlchemyComponentsReady()

-- параметры
нет

-- возвращаемые значения
boolean - true, если компонентов достаточно

-- пример использования
if avatar.IsAlchemyComponentsReady() do
  avatar.DoAlchemyReaction()
end
