common.IsTextFiltered( text )

Проверяет, содержатся ли в тексте запрещенные слова.

-- библиотека:
common

-- объявление:
function IsTextFiltered( text )

-- параметры:
text: WString - текст на проверку

-- возвращаемые значения:
boolean - имеет ли переданный текст запрещенные слова

-- примеры:
local isFiltered = common.IsTextFiltered( text )
