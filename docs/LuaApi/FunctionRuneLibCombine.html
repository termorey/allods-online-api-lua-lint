runeLib.Combine( runeId, instrumentId, crystalCrumbBonus, count )

Попытка соединить одинаковые руны и получить руны на уровень выше.

crystalCrumbBonus - бонус в оплате кристаллической крошкой (от -7 до +7): 0 - дефолтная цена, 1 - +10% в крошке, -1 - -10% в крошке.

-- библиотека:
runeLib

-- объявление:
function Combine( runeId, instrumentId, crystalCrumbBonus, count )

-- параметры:
runeId: ObjectId - идентификатор руны в контейнере
instrumentId: ObjectId - идентификатор инструмента в контейнере
crystalCrumbBonus : number (integer) - бонус в оплате кристаллической крошкой
count: number (integer) - требуемой количество новых рун

-- возвращаемые значения:
нет

-- пример:
runeLib.Combine( runeId, instrumentId, 3 )
