userMods.ToWString( sysText )

Возвращает локализуемую строку по обычной.

Функцию можно использовать только в пользовательских дополнениях.

-- библиотека:
userMods

-- объявление:
function ToWString( sysText )

-- параметры:
sysText: string - исходная строка

-- возвращаемые значения:
WString - исходная строка, преобразованная в локализуемый текст

-- примеры:
local text = userMods.ToWString( "Debug information" )
