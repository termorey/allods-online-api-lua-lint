lifestyle.CanChangePreset()

Возвращает true, если сейчас возможно переключить пресет стиля.

-- библиотека:
lifestyle

-- объявление:
function CanChangePreset()

-- параметры:
нет

-- возвращаемые значения:
boolean - true, если переключение пресетов доступно

-- пример:
if lifestyle.CanChangePreset() then
  lifestyle.SetActivePreset( 2 )
end
