На главную

avatar.CanImproveInnateStats()

Показывает, можно ли распределять свободные пункты характеристик. При изменении этого состояния приходит событие EVENT_AVATAR_STATS_CAN_IMPROVE_CHANGED.

-- библиотека:
avatar

-- объявление:
CanImproveInnateStats()

-- параметры:
нет

-- возвращаемые значения:
boolean - true, если можно распределять свободные пункты характеристик


-- пример:
local canImproveInnateStats = avatar.CanImproveInnateStats()
