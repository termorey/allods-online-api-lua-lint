Пользовательские дополнения

На главную

unit.GetMana( unitId )

Возвращает информацию об мане персонажа. Если значения каких-то параметров неизвестны, возвращается nil.

-- библиотека:
unit

-- объявление:
function GetMana( unitId )

-- параметры:
unitId: ObjectId (not nil) - идентификатор юнита

-- возвращаемые значения:
table, поля:
mana: number (int) or nil - текущая мана юнита для дружественных целей(аватар, группа), в остальных случаях присылается nil
maxMana: number (int) or nil - максимальная мана юнита для дружественных целей(аватар, группа), в остальных случаях присылается nil
percents: number (int) or nil - процент текущей маны от максимальной

-- пример:
unit.GetMana( unitId )
