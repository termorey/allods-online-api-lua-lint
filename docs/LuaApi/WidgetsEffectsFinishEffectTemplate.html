На главную

FinishEffect( self, finalPosition, sendEvent )

Шаблон функции прерывания эффекта / цепочки эффектов на виджете. Является абстрактным и не может быть вызван непосредственно.

Типы эффектов

-- Класс
WidgetSafe или один из его наследников (зависит от типа эффекта)

-- Семантика
function FinishEffect( self, finalPosition, sendEvent )

-- Входные параметры
self: WidgetSafe - Виджет на котором прерывается эффект
finalPosition: boolean or nil - Выставить состояние виджета как конечное из эффекта / цепочки эффектов (true) или оставить виджет в том состоянии в котором он находится в момент прерывания эффекта (false). По умолчанию false.
sendEvent: boolean or nil - Послать событие об окончании эффекта / цепочки эффектов (true) или не посылать (false). По умолчанию false.
  Если задано true то событие будет послано вне зависимости от того было ли оно затребовано при старте эффекта / цепочки эффектов.

-- Возвращаемые значения
Нет
