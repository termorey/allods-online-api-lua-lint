Remove( self, widget )

Удаление указанного элемента (по виджету). Отсутствие в контейнере данного виджета считается ошибкой.

ВАЖНО: Удаляется именно элемент контейнера, но не его WidgetSafe. Он будет возвращен владельцу (будет неявно вызван Owner:AddChild( widget ) )

-- класс
ScrollableContainerSafe

-- семантика
function Remove( self, widget )

-- входные параметры:
widget: WidgetSafe - удаляемый элемент

-- возвращаемое значение:
нет

-- пример
wtScrollableContainer:Remove( wtButton )
