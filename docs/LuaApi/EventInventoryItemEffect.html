EVENT_INVENTORY_ITEM_EFFECT

Оповещение о начале эффекта для элемента в инвентаре. '''Устарело, в будущем будет удалено из API. Взамен нужно использовать EVENT_CONTAINER_ITEM_EFFECT'''

Типы эффектов: ''' EFFECT_TYPE_...'''. Для некоторых из них в сообщении передаются дополнительные поля (см.''' EFFECT_TYPE_...''' ).

Тип контейнера в ITEM_CONT_....

-- поля:
effect: number (enum EFFECT_TYPE_...) - тип начавшегося эффекта
slot: number (int)  - слот
slotType: number (enum ITEM_CONT_...) - тип контейнера

для EFFECT_TYPE_COOLDOWN_STARTED:
duration: number(int) - общая продолжительность
remaining: number(int) - оставшееся время
