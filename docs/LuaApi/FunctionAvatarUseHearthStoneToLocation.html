На главную

avatar.UseHearthStoneToLocation()

Телепортирует игрока в заданную точку. Идентификатор точки телепортации должен быть взят из списка точек телепортации, имеющихся у аватара (см avatar.GetTeleportLocations()) или из привязки HearthStone (см avatar.GetHearthStoneLocator(), поле locationId).

-- библиотека:
avatar

-- объявление:
function UseHearthStoneToLocation( locationId )

-- параметры
locationId: ObjectId - идентификатор точки телепортации

-- возвращаемые значения
нет

Пример:

avatar.UseHearthStoneToLocation( locationId )

См. также avatar.GetTeleportLocations() и avatar.GetHearthStoneLocator().
