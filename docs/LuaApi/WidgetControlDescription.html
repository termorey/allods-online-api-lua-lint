Контролы

Контролы в Lua представлены как объекты, функциональность которых реализована в коде.

-- правильный вызов (короткая форма)
mainForm:Show( true )

-- неправильная запись, хотя и допустимая в Lua (полная форма)
mainForm.Show( mainForm, true )
