PROGRESSIVE_DOWNLOAD_ACTIVITY_...

Возможные значения статуса прогрессивной загрузки.

Перечисление определено в коде, значения:

-- значения:
PROGRESSIVE_DOWNLOAD_ACTIVITY_NO - прогрессивная загрузка не производится
PROGRESSIVE_DOWNLOAD_ACTIVITY_LOW - загружаются данные низкой степени важности
PROGRESSIVE_DOWNLOAD_ACTIVITY_MEDIUM - загружаются данные средней степени важности
PROGRESSIVE_DOWNLOAD_ACTIVITY_HIGH - загружаются данные высокой степени важности

См. также:

  • progressiveDownloadLib.GetStatus()

  • EVENT_PROGRESSIVEDOWNLOADING_STATUS_CHANGED
