На главную

avatar.GetCurrencyId(sysName)

Возвращает информацию об альтернативной валюте по её внутриигровому имени.

В случае ошибки возвращается nil. Если валюта в данный момент не найдена у игрока, но в принципе существует в системе, всё равно возвращается nil.

-- библиотека:
avatar

-- объявление:
function GetCurrencyId( sysName )

-- параметры:
sysName: string or nil - внутриигровое имя альтернативной валюты.

-- возвращаемые значения:
CurrencyId or nil - идентификатор ресурса альтернативной валюты.

-- пример:
local myrrhId = avatar.GetCurrencyId( "myrrh" )
if myrrhId then
  local info = myrrhId:GetInfo()
  if info then
    local currentValue = info.value
  end
end
