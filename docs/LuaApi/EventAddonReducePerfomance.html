EVENT_ADDON_REDUCE_PERFOMANCE

Событие присылается когда система фиксирует критически большое ухудшение производительность клиента из-за конкретного аддона.

-- поля:
name: WString - название аддона
sysName: String - системное имя аддона
deltaFps: Number - приблизительное количество кадров в секунду которые отнимает аддон
maxDeltaFps: Number - максимальное количество кадров в секунду которые отнимал аддон
