На главную

enchant.GetItemSlotsInfo( itemId )

Получить информацию о слотах предмета, доступных для вставки энчантов и соответствующих этим слотам статах.

-- библиотека
enchant

-- объявление:
function enchant.GetItemSlotsInfo( itemId )

-- параметры
itemId: ObjectId - идентификатор предмета

-- возвращаемые значения
table или nil - nil если нет информации; иначе таблица, проиндексированная элементами числового множества InnateStats, формат значений:
  table или nil - nil если у предмета нет статов для данного слота, иначе список (индексируется от 0), относящихся к данному слоту статов (числовое множество InnateStats)

-- пример использования
local slots = enchant.GetItemSlotsInfo( itemId )
if slots then
  local defenceSlotStats = availableSlots[ENUM_EnchantSlot_Defence]
end
