На главную

IsMuted( self )

Возвращает true если данный звук заглушен в момент вызова.

Внимание! Возвращается состояние данного конкретного звука. Итоговое состояние может отличаться в зависимости от настройки вышележащих групп звуков.

Примечание - для пользовательских дополнений все звуки относятся к группе Интерфейс, опция interfaceVolume / audio_option_interface_volume

-- класс
Sound2DLua

-- семантика
function IsMuted( self )

-- входные параметры
self - Sound2DLua - экземпляр Sound2DLua

-- возвращаемое значение
bool - true если звук заглушен

-- пример
if sound:IsMuted() then
    LogInfo( "Sound is muted" )
end
