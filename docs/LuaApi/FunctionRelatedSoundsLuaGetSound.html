RelatedSoundsLua:GetSound( self, sysName )

Возвращает один из звуков группы по его идентификатору.

RelatedSoundsLua

function GetSound( self, sysName )

-- параметры:

-- возвращаемые значения:

-- пример:
local Sound2D = group:GetSound( "test01" )
