WString:ToInt( self )

Конвертирует строку в целое число, если парсинг не удался возвращает nil.

-- класс
WString

-- семантика
function ToInt( self )

-- входные параметры:
self: WString (not nil) - объект WString

-- возвращаемое значение:
number (int) or nil - результат конвертации (nil при невозможности конвертации)

-- примеры:
ToWs( "1234" ):ToInt()
  -- 1234
ToWs( "text" ):ToInt()
  -- nil
