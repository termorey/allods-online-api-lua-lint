ForceReposition( self, force )

Вызывает немедленную репозицию контейнера. По умолчанию репозиция выполняется 1 раз в конце кадра если была запланирована. К планированию репозиции приводит

  • Добавление или удаление элементов

  • Изменение placement контейнера, любого из его родителей или детей (в том числе элементов).

Если после такого действия необходимо сразу получить актуальные placement контейнера или его элементов - нужно вызвать данный метод.

Важно! Репозиция контейнера с большим числом элементов - дорогая операция, не следует вызывать ее без существенной необходимости.

Важно! Флаг force - жуткое колдунство. Если он вам понадобился, то что-то пошло сильно не так.

-- класс
TextContainerSafe

-- семантика
function ForceReposition( self )

-- входные параметры:
self - TextContainerSafe - целевой контейнер
force - bool (default false) or nil - выполнить репозицию даже если она не запланирована

-- возвращаемое значение:
Нет

-- пример
wtTextContainer:ForceReposition( )
