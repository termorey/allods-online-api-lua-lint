На главную

ButtonSafe:SetTextColor( self, tagName, color, <colorType> )

Используется для установки цвета всего текста или только с именем tagName.

DEPRECATED: Функция устарела и помечена к удалению. Используйте стили (предпочтительно) или SetTextAttributes

-- класс
ButtonSafe

-- семантика
function SetTextColor( self, tagName, color, [colorType] )

-- входные параметры
tagName: WString or string - имя тега, если nil, то красится весь текст
color: integer or string "aarrggbb" or wstring "aarrggbb" or table Color - новый цвет текста. Если nil, то установленный цвет сбрасывается
colorType: EnumColorType - определяет для чего устанавливается цвет тект/тень/аутлайн. Если данный параметр отсутствует, то цвет применяется для текста.

-- возвращаемое значение


-- пример
wtButton:SetTextColor( debugCommon.ToWString( "name1" ), { a = 1, r = 1, g = 1, b = 0 }, ENUM_ColorType_SHADOW )

см. [:EnumColorType]
