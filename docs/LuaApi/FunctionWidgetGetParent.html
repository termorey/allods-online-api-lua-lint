GetParent( self )

Возвращает родительский виджет для всех элементов, кроме форм. Для форм всегда возвращается nil. Если родителя нет, то возвращается nil.

Уточнение: возращается именно родительский виджет, а не владелец (owner).

-- класс
метод доступен у всех типов виджетов

-- семантика
function GetParent( self )

-- входные параметры
нет дополнительных параметров

-- возвращаемое значение
WidgetSafe/nil - родительский виджет или nil, если родителя нет или текущий виджет является формой

-- пример
local wtParent = widget:GetParent()
if wtParent then
 LogInfo( "parent name: ", wtParent:GetName() )
end
