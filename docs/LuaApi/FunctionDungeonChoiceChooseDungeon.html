На главную

dungeonChoice.ChooseDungeon( dungeonIndex, rewardIndex )

Выбрать личное приключение для атаки из списка dungeonChoice.GetList() по индексу. Также надо выбрать награду по индексу.

-- библиотека:
dungeonChoice

-- объявление:
function ChooseDungeon( dungeonIndex, rewardIndex )

-- параметры:
dungeonIndex: number (integer) - индекс личного приключения в списке
rewardIndex: number (integer) - индекс награды, которую выбирает игрок

-- возвращаемое значение:
нет

-- пример:
local list = dungeonChoice.GetList()
if list[ 0 ] then
  dungeonChoice.ChooseDungeon( 0, 0 )
end
