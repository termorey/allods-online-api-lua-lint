avatar.DuplicateMetaEnhancerItem( itemId, instrumentId )

Копировать метаитем-улучшитель применением на нём предмета-инструмента.

-- библиотека:
avatar

-- объявление:
function DuplicateMetaEnhancerItem( itemId, instrumentId )

-- параметры:
itemId: ObjectId - идентификатор конвертируемого метапредмета в контейнере; должен быть метапредметом-улучшителем (иное считается ошибкой и команда игнорируется),
                   проверяется с помощью avatar.GetItemMetaInfo( itemId ) по флагу isMetaEnchancer
instrumentId: ObjectId - идентификатор инструмента в контейнере; должен быть специализированным преметом-дубликатором, иное считается ошибкой (иное считается ошибкой и команда игнорируется),
                       проверяется с помощью avatar.IsMetaEnhancerDuplicator( itemId )

-- возвращаемые значения:
нет

-- пример:
if avatar.IsMetaEnhancerDuplicator( instrumentId ) then
  avatar.DuplicateMetaEnhancerItem( itemId, instrumentId )
end

См. также avatar.IsMetaEnhancerDuplicator( itemId ).
