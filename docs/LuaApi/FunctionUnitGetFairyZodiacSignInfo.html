Пользовательские дополнения

На главную

unit.GetFairyZodiacSignInfo( unitId )

Возвращает информацию о знаке зодиака покровителя указанного юнита.

-- система:
UIState/GameState
функция имеет основную реализация в логике и расширенную реализацию в UI

-- библиотека:
unit

-- объявление:
function GetFairyZodiacSignInfo( unitId )

-- параметры:
unitId: ObjectId (not nil) - идентификатор юнита

-- возвращаемые значения:
table or nil - если информация найдена, то таблица с полями:
  sysName: string - уникальное системное имя знака зодиака
  name: WString - имя знака зодиака
  description: WString - описание знака зодиака
  --{ часть таблицы, реализованная только для UI
  image: TextureId - идентификатор текстуры для иконки
  --}

-- пример:
local info = unit.GetFairyZodiacSignInfo( unitId )
if info then
  LogInfo( info.name )
end
