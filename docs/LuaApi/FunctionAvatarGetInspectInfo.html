На главную

avatar.GetInspectInfo()

Возвращает информацию об умении игрока просматривать вещи другого игрока и текущих просматриваемых объектах.

ВНИМАНИЕ! Для сторонних аддонов есть ограничения:

1. Этот метод недоступен на PvP активностях, в которых есть возможность голосовать за изгнание игрока. Проверить доступность можно функцией avatar.IsInspectAllowed()

-- библиотека:
avatar

-- объявление:
function GetInspectInfo()

-- параметры:
нет

-- возвращаемые значения:
table - таблица с полями:
  rank: number (int) or nil - ранг умения игрока подсматривать чужие вещи, если оно у него есть
  playerId: ObjectId or nil - идентификатор инспектируемого игрока
  mountId: ObjectId or nil - идентификатор маунта инспектируемого игрока

-- пример:
local info = avatar.GetInspectInfo()
local rank = info.rank
if info.mountId then
  local mountInfo = mount.GetInfo( info.mountId )
end
