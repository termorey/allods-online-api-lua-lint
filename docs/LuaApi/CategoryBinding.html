CategoryBinding

Функции и события Lua API библиотеки binding - работа с горячими клавишами. Предназначены для реализации работы в опциях с горячими клавишами, однако, работают по независимому, собственному механизму в виду значительной уникальной функциональности.

Для изменения игроком предназначены только заранее заданные (и локализованные) горячие клавиши. Остальная часть настроек через этот механизм недоступна. Список биндов задается через базу. Бинды организованы в секции (обычно по темам), для каждого бинда (игровой реакции) может быть несколько (штатно: 0..2) горячих клавиш.

Замечание. При работе с биндами через функции библиотеки binding значения, выставленные через конфигурационный файл (input.cfg) будут сброшены (удалены) для всех секций, прописанных в базе.

Все функции и события для работы с биндингом реализованы в UI.

Дополнительные подробности: LuaApiDetails (закрытая ссылка)
