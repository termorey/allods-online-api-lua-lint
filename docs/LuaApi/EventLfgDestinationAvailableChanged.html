EVENT_LFG_DESTINATION_AVAILABLE_CHANGED

Событие посылается при изменении доступности для игрока целевой локации, т.е. локация стала доступна, либо наоборот не доступна по условиям входа.

-- поля:
destination: LfgDestinationId - идентификатор целевой локации

Связанные ссылки:

  • LfgDestinationId

  • lfgLib.IsDestinationAvailable( destinationId )

  • lfgLib.GetDestinationRequirements( destinationId )
