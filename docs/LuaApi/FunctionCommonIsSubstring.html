На главную

common.IsSubstring( text1, text2 )

Возвращает true, если вторая строка является подстрокой первой с учётом регистра.

DEPRECATED: Функция устарела и помечена к удалению. Используйте метод WString IsContain( other, ignoreRegistry )

-- библиотека:
сommon

-- объявление:
function IsSubstring( text1, text2 )

-- параметры:
text1: WString - первый локализуемый текст
text2: WString - второй локализуемый текст

-- возвращаемые значения:
boolean - true, если вторая подстрока является подстрокой первой строки

-- примеры:
local result = common.IsSubstring( debugCommon.ToWString( "Text1" ), debugCommon.ToWString( "ext" ) )
