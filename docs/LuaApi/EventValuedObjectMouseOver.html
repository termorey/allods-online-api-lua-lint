EVENT_VALUED_OBJECT_MOUSEOVER

Уведомление, что был на ValuedObject (объект представляющий игровой объект в UI тексте) был наведён или был убран с него курсор мыши.

-- поля:
object: ValuedObject - кликнутый объект в тексте
cursorPosX: number( integer ) - X позиция курсора, в которой произошёл клик
cursorPosY: number( integer ) - Y позиция курсора, в которой произошёл клик
kbFlags: number (enum KBF_...) - состояние модификаторов клавиатуры (ссылка ниже)
mouseButton: number (enum MOUSE_BUTTON_...) - какая кнопка мыши была нажата
widget: WidgetSafe - виджет в котором находится ValuedObject
isActive: boolean - true, если курсор мыши был наведён на объект. Если курсор был убран с объекта, то false

См. KBF_...

См. MOUSE_BUTTON_...
