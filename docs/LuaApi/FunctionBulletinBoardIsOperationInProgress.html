bulletinBoard.IsOperationInProgress()

Возвращает признак, выполняется ли операция с доской объявлений: размещение нового, редактирование или удаление существующего объявления. Начинать новую операцию до окончания предыдущей нельзя.

-- библиотека:
bulletinBoard

-- объявление:
function IsOperationInProgress()

-- параметры:
нет

-- возвращаемые значения:
boolean - выполняется ли операция в данный момент

-- примеры:
if bulletinBoard.IsOperationInProgress() then
  -- запретить редактирование и удаление объявлений
end
