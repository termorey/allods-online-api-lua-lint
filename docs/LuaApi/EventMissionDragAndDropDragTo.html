EVENT_DND_DRAG_TO

Событие приходит при перемещении курсора во время drag&drop.

Замечание. Событие может прийти после окончания drag&drop, если оно было отправлено в очередь до уведомления системы об окончании перетаскивания. Эту ситуацию надо проверять дополнительно.

-- поля:
posX: number (int) - координата курсора по X (реальная)
posY: number (int) - координата курсора по Y (реальная)
outside: boolean - находится ли курсор за пределами окна приложения
kbFlags: number (enum KBF_...) - состояние модификаторов клавиатуры (ссылка ниже)

См. KBF_...
