avatar.GetReputationValue( reputationLevel )

Получить значение репутации соответствующее уровню репутации.

-- библиотека:
avatar

-- объявление:
function GetReputationValue( reputationLevel )

-- параметры:
reputationLevel: number (enum REPUTATION_LEVEL_...) - уровень репутации

-- возвращаемые значения
Number(int) or nil - значение репутации для получения заданного уровня репутации; nil - если уровень репутации задан некорректно

-- пример
local reputationValue = avatar.GetReputationValue( REPUTATION_LEVEL_FRIENDLY )
