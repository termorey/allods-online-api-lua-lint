KBF_...

Числовое перечисление. Состояние модификаторов клавиатуры. Допустима комбинация нескольких модификаторов зажатых клавиш.

KBF_NONE - нет модификаторов
KBF_SHIFT - зажата одна из клавиш "Shift"
KBF_ALT - зажата одна из клавиш "Alt"
KBF_CTRL - зажата одна из клавиш "Ctrl"
KBF_ANY - признак, что модификатор не имеет значения

-- примеры:
if bit.band( params.kbFlags, KBF_SHIFT ) ~= 0 then
  LogInfo( "Shift pressed" )
end
