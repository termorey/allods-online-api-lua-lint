craft.GetReforgeRecipe()

Возвращает текущий рецепт и итем для reforge крафта.

-- библиотека
craft

-- объявление:
function GetReforgeRecipe()

-- параметры
нет

-- возвращаемые значения
table or nil - в случае, если рецепт не был установлен возвращается nil, иначе таблица с полями:
  recipe - ReforgeResourceId, идентификатор ресурса для reforge крафта
  itemId - ObjectId, идентификатор итема для reforge крафта

-- пример использования
LogInfo( craft.GetReforgeRecipe() )
