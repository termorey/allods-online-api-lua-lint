poweredLSWeaponsLib.GetEventItemsList()

Возвращает индексированный с 1 cписок предметов прокачанного оружия для соответствующего рула (эвента)

-- библиотека:
poweredLSWeaponsLib

-- объявление:
GetEventItemsList()

-- параметры:
нет

-- возвращаемые значения:
table of ObjectId or nil - если есть рул (эвент) для прокачанного оружия, то индексированный с 1 cписок предметов прокачанного оружия.

Пример:

local itemsIds = poweredLSWeaponsLib.GetEventItemsList()
