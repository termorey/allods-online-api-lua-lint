На главную

mentor.GetPlayerReward( mentorPlayerId )

Возвращает подробное описание наград, которые дает (будет периодически давать) или уже дал ученик - субъект менторских взаимоотношений.

-- библиотека:
mentor

-- объявление:
function GetPlayerReward( mentorPlayerId )

-- параметры:
mentorPlayerId: ObjectId - идентификатор субъекта менторских взаимоотношений

-- возвращаемые значения -
table or nil - если не произошло ошибки и субъект действительно ученик, то таблица с полями:
  moneyPercent: number (integer) - сколько процентов заработаных денег ментор дополнительно получает от ученика
  realMoney: number (integer) - сколько процентов заработаных денег ментор уже получил от ученика
  items: table of table - индексированная с 0 таблица описаний предметов, определенный процент которых получает ментор
    itemId: ObjectId - идентификатор предмета
    percent: number (integer) - сколько процентов таких предметов получает ментор при получении предметов учеником
    realCount: number (integer) - сколько таких предметов уже получил ментор при получении предметов учеником

-- пример:
local info = mentor.GetPlayerReward( mentorPlayerId )
