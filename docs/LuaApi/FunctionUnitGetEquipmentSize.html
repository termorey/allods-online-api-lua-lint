unit.GetEquipmentSize( unitId )

Возвращает количество слотов под экипировку у данного юнита. Юнит может быть или главным игроком или игроком, инспектируемым в данный момент.

-- библиотека:
unit

-- объявление:
function GetEquipmentSize( unitId )

-- параметры:
unitId: ObjectId (not nil) - идентификатор юнита

-- возвращаемые значения:
number (int) - число слотов под экипировку юнита

-- пример:
unit.GetEquipmentSize( unitId )
