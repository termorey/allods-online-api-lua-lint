Пользовательские дополнения

На главную

unit.SetPlayerTitle( unitId, titleId )

Выбирает (устанавливает текущий) титул персонажа. Титул можно устанавливать только собственному персонажу

-- библиотека:
unit

-- объявление:
function SetPlayerTitle( unitId, titleId )

-- параметры:
unitId: ObjectId (not nil) -- идентификатор персонажа
titleId: ObjectId or nil - идентификатор титула (идентификатор), или nil для сброса титула

-- возвращаемые значения:
result: boolean - true, в случае удачи, иначе - false

Пример:

local titles = unit.GetPlayerTitles( unitId )
if titles then
  unit.SetPlayerTitle( unitId, titles[ 1 ].id )
end

''См. также: EVENT_PLAYER_TITLE_CHANGED''
