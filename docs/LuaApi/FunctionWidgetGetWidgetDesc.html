GetWidgetDesc( self )

Возвращает описатель виджета в ресурсной системе. Работает только для виджетов, созданных на основе таких описателей (большинство виджетов именно такие, исключения будут описываться отдельно).

Функция вернет ошибку если выполняется любое из нижележащих условий

  • Dиджет является внутренним

  • Dиджет является формой

-- класс
метод доступен у всех типов виджетов (WidgetSafe)

-- семантика
function GetWidgetDesc( self )

-- входные параметры
нет

-- возвращаемое значение
WidgetDesc - описатель виджета из ресурсной системы

-- пример
local desc = wtTemplate:GetWidgetDesc()
