На главную

mentor.GetPredictedCooldownsInfo()

Возвращает предсказанние о кулдаунах на запрет каких-либо действий в менторстве. Для каждого запрета возвращается количество миллисекунд от начала действия до окончания запрета на это действие. Если запрета не будет, то для данного действия возвращается 0.

-- библиотека:
mentor

-- объявление:
function GetPredictedCooldownsInfo()

-- параметры:
нет

-- возвращаемые значения -
table or nil - если не произошло ошибки, то таблица с полями:
mentorRefuseRemainingMs: number (integer) - сколько миллисекунд будет длиться запрет на отказ от ментора
apprenticeRefuseRemainingMs: number (integer) - на отказ от ученика
mentorFlagRemainingMs: number (integer) - на сброс флага ментора
apprenticeFlagRemainingMs: number (integer) - на сброс флага ученика


-- пример:
local info = mentor.GetPredictedCooldownsInfo()
LogInfo( info.mentorRefuseRemainingMs )
