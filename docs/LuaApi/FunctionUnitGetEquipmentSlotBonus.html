На главную

unit.GetEquipmentSlotBonus( unitId, slot, slotType )

Возвращает таблицу с информацией о бонусах, даваемых предметом в определенном слоте экипировки данного юнита. Юнит может быть или главным игроком или игроком, инспектируемым в данный момент.

ВАЖНО! Информация возвращается с учетом наложенных бафов, влияющих на характеристики экипировки.

Функция возвращает результаты в том же формате что и itemLib.GetBonus( itemId )

См. DRESS_SLOT_...

Тип контейнера в ITEM_CONT_...

-- библиотека:
unit

-- объявление:
function GetEquipmentSlotBonus( unitId, slot, slotType )

-- параметры:
unitId: ObjectId (not nil) - идентификатор юнита
slot: number (enum DRESS_SLOT_...) - индекс слота (0..DRESS_SLOT_UNDRESSABLE - 1)
slotType: number (enum ITEM_CONT_...) - тип контейнера

-- возвращаемые значения:
см. itemLib.GetBonus( itemId )

-- пример:
local bonus = unit.GetEquipmentSlotBonus( unitId, slot, ITEM_CONT_EQUIPMENT )
if bonus then
  local power = bonus.miscStats.power.effective
end
