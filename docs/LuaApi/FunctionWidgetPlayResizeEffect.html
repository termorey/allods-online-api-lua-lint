PlayResizeEffect( self, start, finish, timeMs, algoDesc, needEvent, eventReceivers )

Запускает эффект изменения размера виджета. Имеет тип ET_RESIZE, соответствует шаблону PlayEffect( self, start, finish, timeMs, algoDesc, needEvent, eventReceivers )

Типы start / finish: table (WidgetPlacement), значимые поля sizeX, sizeY

Замечание: Используются ресурсные поля размера, если виджет выровнен по ширине (WIDGET_ALIGN_BOTH) или имеет неявное масштабирование (WIDGET_SIZING_CHILDREN / WIDGET_SIZING_INTERNAL) то данный эффект работать не будет.

-- Класс
WidgetSafe

-- Типы шаблонных аргументов
start: table (WidgetPlacement) or nil - начальный размер виджета
finish: table (WidgetPlacement) or nil - конечный размер виджета

-- пример
wtTestButton:PlayResizeEffect( nil, { sizeX = 0 }, 2000, EA_MONOTONOUS_INCREASE )
