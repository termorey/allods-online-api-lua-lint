EVENT_GUILD_AVATAR_HISTORY

Серверное событие с гильдейской банковской историей аватара. Приходит после вызова функции guild.GetAvatarHistory().

-- Параметры:
table of table - индексированная с нуля таблица с описаниями конкретных операций, каждая из которых представлена таблицей с полями:
  moneyChange: number (integer) - величина транзакции в меди
  comment: WString - комментарий при проведении операции
  time: LuaFullDateTime - время транзакции
  changeReason: number (enum ENUM_MoneyChangeReason_...) - причина транзакции
  sysChangeReason: string (enum "ENUM_MoneyChangeReason_...") - причина транзакции

См. LuaFullDateTime, EnumMoneyChangeReason, EnumTabardType
