depositeBox.Move( depositeSlotFrom, depositeSlotTo, countNoMore )

Попытка переместить предметы в хранилище.

-- библиотека:
depositeBox

-- объявление:
function Move( depositeSlotFrom, depositeSlotTo, countNoMore )

-- параметры:
depositeSlotFrom: number( int ) - индекс исходного слота с предметами в хранилище [0..]
depositeSlotTo: number( int ) - индекс целевого слота в хранилище [0..]
countNoMore: number( int ) or nil - максимальное количество предметов из пачки для переноса или nil, если нужно перенести всю пачку

-- возвращаемые значения:
нет

-- пример:
depositeBox.Move( 0, 1, nil )
