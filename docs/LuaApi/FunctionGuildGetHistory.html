На главную

guild.GetHistory( page, orderBy, rearOrder )

Запрашивает у сервера историю операций казны гильдии постранично. В ответ придет событие EVENT_GUILD_BANK_OPERATION_RESULT с параметром sysOperationType равным "ENUM_GuildBankOperation_GetHistory". Успешным итогом вызова данной функции является вызов события EVENT_GUILD_BANK_HISTORY_CHANGED.

-- библиотека
guild

-- объявление:
function GetHistory( page, orderBy, rearOrder )

-- параметры
page: number(int) - интересующая страница(начиная с 1)
orderBy: number(ENUM_GuildHistoryOrder) - сортировать историю по параметру
rearOrder: boolean - задает порядок сортировки; true - обратный порядок, false - прямой

-- возвращаемые значения
нет

-- пример использования
guild.GetHistory( 3, ENUM_GuildHistoryOrder_OPERATIONTIME, false ) -- выдать третью страницу истории операций гильдии, сортируя операции по времени в прямом порядке
