Пользовательские дополнения

На главную

options.GetCollectionInfo( collectionId )

Возвращает информацию о данной коллекции опций (странице, группе или блоке).

-- библиотека:
options

-- объявление:
function GetCollectionInfo( collectionId )

-- параметры:
collectionId: ObjectId - идентификатор коллекции опций

-- возвращаемые значения:
таблица с полями:
isEnabled: boolean - можно ли менять коллекцию и ее элементы
isNeedPreview: boolean - нужно ли использовать режим предварительного просмотра
isPreview: boolean - находится ли какой-нибудь элемент коллекции в режиме предварительного просмотра
sysCustomType: string - уникальный идентификатор коллекции для возможной специальной обработки
name: название коллекции
description: описание/тултип/комментарий и т.п. коллекции

-- пример:
local collectionInfo = options.GetCollectionInfo( collectionId )
