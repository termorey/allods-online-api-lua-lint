GetPosObjectCount( self )

Возвращает допустимое количество объектов для задания позиции курсора в EditLine. Допустимые позиции курсора лежат в диапазоне [0..GetPosObjectCount()].

-- класс
EditLineSafe

-- семантика
function GetPosObjectCount( self )

-- входные параметры
нет

-- возвращаемое значение
number (int) - количество объектов (оно же последний допустимый индекс для курсора)

-- пример
local posObjectCount = wtEditLine:GetPosObjectCount()
wtEditLine:SetPos( posObjectCount ) -- поставим курсор в конец EditLine
