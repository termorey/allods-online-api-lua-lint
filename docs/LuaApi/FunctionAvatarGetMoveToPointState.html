На главную

avatar.GetMoveToPointState()

Возвращает информацию о режиме движения в точку для аватара игрока.

-- библиотека:
avatar

-- объявление:
function GetMoveToPointState()

-- параметры:
нет

-- возвращаемые значения:
таблица с полями:
 isMoving: boolean - находится ли персонаж в режиме движения в точку, для режима движения в точку доступны дополнительные поля:
  position: таблица с координатами целевой точки, поля:
   posX: number (double)
   posY: number (double)
   posZ: number (double)

Пример:

local moveToPoint = avatar.GetMoveToPointState()
if moveToPoint.isMoving then
 local position = moveToPoint.position
 LogInfo( "target pos: ", position.posX, "/", position.posY, "/", position.posZ )
end
