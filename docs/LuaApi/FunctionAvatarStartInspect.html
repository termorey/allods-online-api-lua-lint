На главную

avatar.StartInspect( unitId )

Попытка инспектировать игрока. Пользователь может получать экипировку инспектируемой цели.

ВНИМАНИЕ! В некоторых MM активностях данная функция недоступна. Проверить доступность можно функцией avatar.IsInspectAllowed()

-- библиотека:
avatar

-- объявление:
function StartInspect( unitId )

-- параметры:
unitId: ObjectId (not nil) -- идентификатор игрока

-- возвращаемые значения:
нет

-- пример:
avatar.StartInspect( unitId )
