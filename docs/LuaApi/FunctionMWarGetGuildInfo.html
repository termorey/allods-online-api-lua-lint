На главную

mwar.GetGuildInfo( guildId )

Информация о гильдии, участвующей в Доминионе. При изменении приходит событие EVENT_MWAR_LADDER_CHANGED

-- библиотека
mwar

-- объявление:
function GetGuildInfo( guildId )

-- параметры
guildId : ObjectId - идентификатор гильдии, можно получить из mwar.GetLadderInfo().guilds

-- возвращаемые значения
table или nil - nil если нет информации; иначе таблица с полями:
  guildId: ObjectId - идентификатор гильдии
  guildName: WString - имя гильдии
  rating: number (integer) - рейтинг гильдии в Доминионе
  symbolics: table - описание символики гильдии; поля такие же, как у mwar.GetGuildSymbolics( guildId )

-- пример использования
local guildInfo = mwar.GetGuildInfo( guildId )
