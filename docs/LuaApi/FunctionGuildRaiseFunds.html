guild.RaiseFunds()

Объявляет сбор в гильдейском банке.

-- библиотека:
guild

-- объявление:
function RaiseFunds( sum, comment, ranks, tabards )

-- параметры:
sum: number (integer) - сумма сбора в меди
comment: WString - комментарий при объявлении сбора
ranks: table of number (integer) - индексы рангов гильдии, попавших в транзакцию. Может быть пустым
tabards: table of number (enum ENUM_TabardType_...) - накидки гильдии, носители которых попали в транзакцию. Может быть пустым

-- возвращаемое значение
нет

-- пример:
guild.RaiseFunds( 10000, comment, {}, {} )
