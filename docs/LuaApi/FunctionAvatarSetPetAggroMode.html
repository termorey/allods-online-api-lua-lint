avatar.SetPetAggroMode( mode )

Установливает для пета аватара тип поведения:

PET_AGGRO_AGGRESIVE - агрессивный,
PET_AGGRO_DEFENSIVE - защитный,
PET_AGGRO_PASSIVE - пассивный.


-- библиотека
avatar

-- объявление:
function SetPetAggroMode( mode )

-- параметры
mode: number ( enum PET_AGGRO_XXX ) - тип поведения

-- возвращаемые значения
нет

-- пример использования
avatar.SetPetAggroMode( PET_AGGRO_DEFENSIVE )
