EVENT_DND_PICK_ATTEMPT

Событие приходит при попытке системы начать drag&drop. Для подтверждения начала перетаскивания скрипт должен вызвать функцию DNDConfirmPickAttempt. В случае подтверждения перетаскивания необходимо запоминать в скрипте идентификатор, пришедший в этом сообщении (повторно он не пересылается).

-- поля:
srcWidget: WidgetSafe - окно, которое пытаются начать перетаскивать
srcId: number (int) - идентификатор, зарегистрированный для окна, которое пытаются начать перетаскивать
posX: number (int) - координата курсора по X (реальная)
posY: number (int) - координата курсора по Y (реальная)
kbFlags: number (enum KBF_...) - состояние модификаторов клавиатуры (ссылка ниже)

См. KBF_...
