SetContainerOffset( self, offset )

Изменяет значение прокрутки контейнера. Может использоваться для восстановления значения прокрутки контейнера после его изменения.

-- класс
ScrollableContainerSafe

-- семантика
function SetContainerOffset( self, offset )

-- входные параметры:
offset: number (float) - величина прокрутки

-- возвращаемое значение:
нет

-- пример:

-- store offset
local offset = wtScrollableContainer:GetContainerOffset()

-- change container
...

-- restore offset
wtScrollableContainer:SetContainerOffset( offset )
