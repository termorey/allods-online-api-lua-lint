guildHallLib.GetQuests()

Возвращает список заданий для прогресса оплота гильдии.

-- библиотека:
guildHallLib

-- объявление:
function GetQuests()

-- параметры:
нет

-- возвращаемое значение:
table or nil -- список заданий, индексация с 1, каждый элемент таблица с полями:
  questId: QuestId -- идентификатор ресурса задания
  count: Number(int) -- счетчик выполнения данного заданий, значения: 0 - если не выполненно, 1 и более если выполненно
  texture: TextureId or nil -- идентификатор текстуры миниатюры задания
  requiredItems: table of ItemId or nil -- список особых предметов необходимых для выполнения задания, индексация с 0

-- пример:
local guildHallQuests = guildHallLib.IsExist() and guildHallLib.GetQuests()
local firstGuildQuest = guildHallQuests and guildHallQuests[1]
local questReward = firstGuildQuest and guildHallLib.GetQuestReward( firstGuildQuest.questId )

См. также:

  • guildHallLib.GetQuestInfo( questId )

  • guildHallLib.GetQuestReward( questId )
