На главную

enchant.GetCompatibleQualities( itemId )

Получить информация о том в предметы какого качества можно вставить данный энчант.

-- библиотека
enchant

-- объявление:
function enchant.GetCompatibleQualities( itemId )

-- параметры
itemId: ObjectId - идентификатор энчанта

-- возвращаемые значения
table of Enum(ITEM_QUALITY) or nil - список типов качества предмета; индексация таблицы начинается с 1

-- пример использования
local qualities = enchant.GetCompatibleQualities( itemId )
