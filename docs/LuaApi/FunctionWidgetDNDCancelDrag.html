DNDCancelDrag()

Функция должна вызываться скриптом для безусловного прекращения drag&drop по каким-либо внешним причинам (например, при переходе в другой режим интерфейса). В момент отмены drag&drop в системе скрипт должен также позаботиться об отмене его логики и визуализации (убрать курсор, подсветку с окон и т.п.).

Замечание: Настоятельно рекомендуется вызывать данную функцию именно в том виджете, DnD которого активно в данный момент. В текущей реализации функция может быть вызвана из любого виджета с одинаковым эффектом, однако такое поведение не гарантируется в будущем.

метод доступен для всех типов виджетов

нет
