На главную

guildHallLib.GetStatesInfo()

Возвращает список состояний оплота гильдии и их описание.

-- библиотека:
guildHallLib

-- объявление:
function GetStatesInfo()

-- параметры:
нет

-- возвращаемое значение:
table or nil -- список состояний, индексация с 1, каждый элемент таблица с полями:
  description: WString -- художественное описание данного состояния
  requiredExp: Number(int) -- требуемое количество прогресса для достижения данного состояния

-- пример:
local guildHallStates = guildHallLib.IsExist() and guildHallLib.GetStatesInfo()
