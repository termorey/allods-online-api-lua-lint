Пользовательские дополнения

На главную

customizationLib.InstallItem( itemId )

Во время кастомизации при работе с активным слотом помещает предмет из слота в контейнере игрока в кастомизационный слот.

-- библиотека:
customizationLib

-- объявление:
function InstallItem( itemId )

-- параметры:
itemId: ObjectId - идентификатор предмета, который надо поместить в слот

-- возвращаемые значения:
нет

-- пример:
customizationLib.InstallItem( itemId )
