itemLib.GetSpell( itemId )

Id спелла, прикреплённого к предмету.

-- библиотека:
itemLib

-- объявление:
GetSpell( itemId )

-- параметры:
itemId: ObjectId  - идентификатор предмета

-- возвращаемые значения:
SpellId или nil - nil если спелл отсутствует, иначе id спелла

Пример:

local spellId = itemLib.GetSpell( itemId )
local spellInfo = spellId and spellLib.GetDescription( spellId )
local spellName = spellInfo and spellInfo.name

См. также:

  • spellLib.GetDescription( id )
