DNDRegisterGeneric( isDragOnly )

Зарегистрировать виджет для drag&drop. Идентификатор будет назначен автоматически. Если идентификатор не важен, то рекомендуется назначать его автоматически. Если виджет разрегистрировать и зарегистрировать снова - будет назначен новый идентификатор. Если это неприемлемо, то надлежит сохранить выданный идентификатор и использовать его при повторной регистрации через DNDRegister

Можно указать в каком режиме для данного контрола может начинаться перетаскивание (drag attempt) - либо только на перетаскивание, либо как на перетаскивание, так и на клик.

-- класс
метод доступен для всех типов виджетов

-- семантика
function DNDRegister( self, isDragOnly )

-- параметры:
self: WidgetSafe - виджет, для которого будет обрабатываться drag&drop
isDragOnly: boolean - drag&drop  включается либо только на перетаскивание, либо как на перетаскивание, так и на клик

-- возвращаемые значения:
number (int) - назначенный уникальный идентификатор, связанный с виджетом wtSrc

-- примеры:
local id = wtPanel:DNDRegisterGeneric( false )
