На главную

unit.GetFairyPowerBonus( unitId )

Возвращает величину прибавки к Могуществу от уровня покровителя. При изменении приходит событие EVENT_UNIT_FAIRY_CHANGED.

-- библиотека:
unit

-- объявление:
function GetFairyPowerBonus( unitId )

-- параметры:
unitId: ObjectId (not nil) - идентификатор юнита

-- возвращаемые значения:
number (integer) or nil - если информация найдена, величина бонуса от уровня покровителя

-- пример:
local powerBonus = unit.GetFairyPowerBonus( unitId )
