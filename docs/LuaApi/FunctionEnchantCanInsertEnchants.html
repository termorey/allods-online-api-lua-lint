На главную

enchant.CanInsertEnchants( itemId )

Показывает, можно ли вообще вставлять хоть какие-нибудь энчанты в предмет.

Примечание: точный ответ, можно ли вставить конкретный энчант, даёт enchant.CanInsertEnchant( enchantItemId, itemId ).

-- библиотека
enchant

-- объявление:
function CanInsertEnchants( itemId )

-- параметры
itemId: ObjectId - идентификатор проверяемого предмета

-- возвращаемые значения
boolean - если true, то в предмет можно вставлять энчанты

-- пример использования
local isEnchantsCarrier = enchant.CanInsertEnchants( itemId )
