Пользовательские дополнения

На главную

avatar.GetEquipmentItemSlot( itemId )

Возвращает слот предмета надетого на главного игрока. Если предмет не надет, то возвращает nil.

См. DRESS_SLOT_....

-- библиотека:
avatar

-- объявление:
function GetEquipmentItemSlot( itemId )

-- параметры:
ObjectId - идентификатор предмета

-- возвращаемые значения:
dressSlot: number (enum) or nil - индекс ячейки в экипировке, см. DRESS_SLOT_... (ссылка выше)

-- пример:
local dressSlot = avatar.GetEquipmentItemSlot( itemId )
