avatar.GetProficiencies()

Возвращает список Id профессий главного игрока. Этот список зависит от класса персонажа и его уровня. В данный момент - это умения персонажа носить определенный класс предметов.

-- библиотека
avatar

-- объявление:
function GetProficiencies()

-- параметры
нет

-- возвращаемые значения
table of ObjectId - список идентификаторов профессий (индексированный с 0)

-- пример использования
local proficiencies = avatar.GetProficiencies()
for i, id in pairs( proficiencies ) do
  local proficiencyInfo = avatar.GetProficiencyInfo( id )
  local name = proficiencyInfo.name
end
