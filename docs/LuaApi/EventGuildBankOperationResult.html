EVENT_GUILD_BANK_OPERATION_RESULT

Присылается, если завершилась операция в гильдейском банке.

См. "ENUM_GuildBankOperation_...".

-- поля
success: boolean - true, если операция завершилась успешно
sysOperationType: string ("ENUM_GuildBankOperation_...") - тип операции, которая завершилась
