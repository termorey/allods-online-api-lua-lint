avatar.GetContextActions( withMount )

Вернуть список идентификаторов контекстных действий. В списке перечислены идентификаторы созданных экземпляров контекстных действий.

Если список меняется, то приходит EVENT_CONTEXT_ACTIONS_CHANGED.

-- библиотека
avatar

-- объявление:
function GetContextActions( withMount )

-- параметры
withMount: boolean or nil - true добавлять контекстные действия маунта; по-умолчанию true

-- возвращаемые значения
table of ObjectId: - список[1, n] идентификаторов экземпляров контекстных действий; где n - число идентификаторов

-- пример использования
local contextActions = avatar.GetContextActions()
for i = 1, #contextActions do
  local actionInfo = avatar.GetContextActionInfo( contextActions[ i ] )
  if actionInfo then
    local enabled = actionInfo.enabled
  end
end
