Пользовательские дополнения

На главную

guild.GetBuffInfo( Id )

Возвращает информацию об указанном бафе гилдии игрока.

Принимает как идентификатор бафа, так и идентификатор ресурса бафа.

Возвращаемые значения аналогичны значениям возращаемым object.GetBuffInfo( Id )

-- система:
UIState/GameState

-- библиотека:
guild

-- объявление:
function GetBuffInfo( Id )

-- параметры:
Id: ObjectId or BuffId - идентификатор или объекта или ресурса буфа

-- возвращаемые значения:
аналогично object.GetBuffInfo( Id )

-- пример:
local id = guild.GetBuffs()[ 0 ]
if id then
  local buffInfo = guild.GetBuffInfo( id )
  if buffInfo then
    local name = buffInfo.name
  end
end
