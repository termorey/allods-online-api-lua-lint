avatar.GetSecrets()

Возвращает список идентификаторов всех тайн мира, которые уже доступны или будут доступны когда-то аватару.

-- библиотека:
avatar

-- объявление:
function GetSecrets()

-- параметры:
нет

-- возвращаемые значения:
table of ObjectId - индексированный с 0 список идентификаторов тайн мира, открытых игроком

-- пример:
local secrets = avatar.GetSecrets()
for i, secretId in pairs( secrets ) do
  local secretInfo = avatar.GetSecretInfo( secretId )
end
