На главную

avatar.GetDestinyPoints()

Возвращает информацию об Очках Судьбы персонажа

-- библиотека:
avatar

-- объявление:
function GetDestinyPoints()

-- параметры:
нет

-- возвращаемые значения:
table, поля:
total: number (int) - накопленные Очки Судьбы персонажа
bonus: number (int) - бонусный пул Очков Судьбы
-- пример:
local destinyPoints = avatar.GetDestinyPoints()

Примечание

Если бонусный пул Очков Судьбы больше 0, то имеется запас "неактивированных" Очки Судьбы, если меньше 0 то имеется бонусный запас, который будет автоматически переводить новые "неактивированные" Очки Судьбы в накопленные.
