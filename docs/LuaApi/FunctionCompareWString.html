Пользовательские дополнения

На главную

common.CompareWString( text1, text2 )

Сравнивает две локализуемые строки с учетом регистра. В случае ошибочных данных (не WString) всегда возвращает -1.

DEPRECATED: Функция устарела и помечена к удалению. Используйте перегруженные операции сравнения

-- библиотека:
сommon

-- объявление:
function CompareWString( text1, text2 )

-- параметры:
text1: WString - первый локализуемый текст
text2: WString - второй локализуемый текст

-- возвращаемые значения:
number (int) - результат сравнения:
 -1 - первая строка лексикографически меньше (либо хотя бы один из аргументов - не WString)
 0 - строки совпадают
 1 - первая строка лексикографически больше

-- примеры:
local result = common.CompareWString( debugCommon.ToWString( "Text1" ), debugCommon.ToWString( "Text2" ) )
