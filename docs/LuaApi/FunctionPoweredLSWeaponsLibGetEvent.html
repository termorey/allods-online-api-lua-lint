poweredLSWeaponsLib.GetEvent()

Возвращает идентификатор рула (эвента) для прокачанного оружия.

-- библиотека:
poweredLSWeaponsLib

-- объявление:
GetEvent()

-- параметры:
нет

-- возвращаемые значения:
ruleId: RuleId или nil - идентификатор рулa (эвента) для прокачанного оружия. В случае отсутствия возможности получить идентификатор (не в миссии) вернётся nil.

Пример:

local ruleId = poweredLSWeaponsLib.GetEvent()
