На главную

avatar.GetTeleportLocationInfo( locationId )

Возвращает информацию о точке телепортации. Информацию можно получить для имеющихся точек телепортации главного игрока, а также точек телепортации мастера телепорта, с которым разговаривает главный игрок, включая точку, в которой стоит мастер.

-- библиотека
avatar

-- объявление
function GetTeleportLocationInfo( locationId )

-- параметры
locationId: ObjectId or TeleportMasterId or table - идентификатор точки телепорта

-- возвращаемые значения
table or nil - если такая точка найдена, то таблица с полями:
  id: ObjectId - идентификатор точки
  name: WString - название точки
  isHighPriority: boolean - эта точка должна показываться перед остальными точками телепортации
  minLevel: int - минимальный комфортный уровень игры (0, если карта не прописана)
  maxLevel: int - максимальный комфортный уровень игры (0, если карта не прописана)

-- пример
local info = avatar.GetTeleportLocationInfo( locationId )
if info then
  local name = info.name
end
