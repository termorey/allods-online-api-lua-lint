ValuedText:ToWString()

Преобразует ValuedText в WString. Сохраняет только видимые символы, вся информация о тегах форматирования будет утеряна.

-- класс
ValuedText

-- семантика
function ToWString( self )

-- параметры:
нет

-- возвращаемые значения:
WString - результирующая строка

-- примеры:
local valuedText = common.CreateValuedText{
    format = wtFormatEdit:GetText(),
    class1 = wtClassEdit:GetText(),
    name1 = wtValueEdit:GetText()
}

local wstr = valuedText:ToWString()
