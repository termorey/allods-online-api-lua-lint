Пользовательские дополнения

На главную

avatar.SetFollowTarget()

Включает/выключает автоматическое движение вслед за объектом, являющимся целью на момент вызова.

-- библиотека:
avatar

-- объявление:
function SetFollowTarget( isFollow )

-- параметры:
isFollow: Boolean - если true - включить режим следования, если false - выключить

-- возвращаемые значения:
нет

Пример:

avatar.SetFollowTarget( true )

См. также EVENT_FOLLOW_MODE_CHANGED.
