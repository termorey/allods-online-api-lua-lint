На главную

rules.GetEvents( getHidden )

Получить список идентификаторов игровых эвентов. Может вернуть пустой список, если он еще не получен с сервера. При первом получении и обновлениях будет приходить EVENT_MISSION_RULES_CHANGED.

-- библиотека:
rules

-- объявление:
function GetEvents( getHidden )

-- параметры:
getHidden: Boolean or nil - возвращать скрытые эвенты, по умолчанию false

-- возвращаемое значение:
table of ObjectId - индексированная с 0 таблица с идентификаторами правил эвента

Пример:

local eventRules = rules.GetEvents()
for index = 0, GetTableSize( eventRules ) - 1 do
  local eventRuleInfo = rules.GetEventInfo( eventRules[ index ] )
end
