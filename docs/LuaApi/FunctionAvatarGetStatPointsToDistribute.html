avatar.GetStatPointsToDistribute()

Даёт информацию о намеченных для распределения пунктах характеристик. Работает только если avatar.CanImproveInnateStats() даёт true.

-- библиотека:
avatar

-- объявление:
GetStatPointsToDistribute()

-- параметры:
нет

-- возвращаемые значения:
table или nil - если нет информации, то nil; иначе таблица с ключами, соответствующими значениям множества InnateStats и значениями (integer), равными количеству пунктов, которые намечены для добавлению к соответствующим характеристикам

-- пример:
local points = avatar.GetStatPointsToDistribute()
