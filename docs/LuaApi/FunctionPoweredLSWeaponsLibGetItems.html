poweredLSWeaponsLib.GetItems()

Возвращает индексированный с 0 список фейковых предметов, которые могут попасть в контейнер прокачанного оружия.

-- библиотека:
poweredLSWeaponsLib

-- объявление:
GetItems()

-- параметры:
нет

-- возвращаемые значения:
table - список, индексация с 1, каждый элемент таблица с полями
  item: ObjectId - идентификатор предмета
  inactiveItem: ObjectId or nil - идентификатор деактивированной версии предмета

Пример:

local items = poweredLSWeaponsLib.GetItems()
local itemInfo = itemLib.GetItemInfo( items[ 1 ].item )
