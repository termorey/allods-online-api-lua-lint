Пользовательские дополнения

На главную

bulletinBoard.GetPost( postId )

Возвращает информацию об указанном объявлении на доске. Если объявление не найдено (например, было удалено), возвращается nil.

-- библиотека:
bulletinBoard

-- объявление:
function GetPost( postId )

-- параметры:
postId: number (int) - идентификатор объявления

-- возвращаемые значения:
таблица с информацией об объявлении или nil, если объявление не найдено, поля таблицы:
 postId: number (int) - идентификатор объявления
 isPremium: boolean - true, если это платное сообщение
 name: WString - имя персонажа
 guild: WString - название гильдии персонажа (если есть)
 isOnline: boolean - находится ли персонаж в игре
 level: number (int) - уровень персонажа
 raceClass: table LuaRaceClassInfoPart or nil - раса-класс персонажа
 sex: table LuaSexInfoPart or nil - описание пола персонажа
 zoneName: WString - название зоны, где находится персонаж
 text: WString - текст объявления, если уже был получен
 lifeTimeMs: number (int64) or nil - время существования объявления в миллисекундах или nil, в случае ошибки
 lifeTime: table or nil - время существования объявления или nil, в случае ошибки. Таблица с полями:
   d, h, m, s: number (integer) - дней, часов, минут, секунд
 remainingTimeMs: number (int64) or nil - оставшееся время существования объявления в миллисекундах или nil, если ограничений по времени нет
 remainingTime: table or nil - оставшееся время существования объявления или nil, если ограничений по времени нет. Таблица с полями:
   d, h, m, s: number (integer) - дней, часов, минут, секунд

-- примеры:
local post = bulletinBoard.GetPost( postId )
if post then
 LogInfo( "name: ", post.name )
end

См. LuaRaceClassInfoPart, LuaSexInfoPart
