userMods.GetAccountUniqueId()

Возвращает уникальный приватный строковый идентификатор аккаунта игрока. Представляет собой строку длинной 32 символа, состоящую из символов шестнадцатеричной системы исчисления, записанных в верхнем регистре.

Пример: 5C976713F32D4D60B51F5281E35CBA6B

-- библиотека:
userMods

-- объявление:
function GetAccountUniqueId()

-- параметры:
нет

-- возвращаемые значения:
String - уникальный приватный строковый идентификатор аккаунта

-- примеры:
local accountUniqueId = userMods.GetAccountUniqueId()
if accountUniqueId == myAccountId then
 -- do something special
end
