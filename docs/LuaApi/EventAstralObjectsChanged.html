EVENT_ASTRAL_OBJECTS_CHANGED

Событие посылается раз в кадр, если за этот кадр в зоне действия аватара появляется или исчезает астральный объект.

-- поля:
spawned: table of ObjectId - индексированная с 0 таблица идентификаторов появившихся объектов
despawned: table of ObjectId - индексированная с 0 таблица идентификаторов исчезнувших объектов
