Пользовательские дополнения

На главную

cartographer.GetSosInfo( id )

Возвращает описание объекта SOS (объект на карте - "призыв о помощи", который появляется при использовании специального фейрверка другим игроком фракции главного игрока). GamePosition - позиция.

-- библиотека:
cartographer

-- объявление:
function GetSosInfo( id )

-- параметры:
id: ObjectId - идентификатор объекта

-- возвращаемые значения
id: ObjectId - идентификатор объекта
durationMs: number (int) - сколько длится показ объекта
remainingMs: number (int) - сколько осталось до окончания показа объекта
requesterName: WString - имя отправителя
zonesMapId: ObjectId or nil - идентификатор интерфейсной карты, где поставлен объект, если она найдена.
position: GamePosition - позиция на указанной карте
message: WString - текст сообщения

Пример:

local sosObjects = cartographer.GetSosObjects()
for i = 0, GetTableSize( sosObjects ) - 1 do
  local sosObject = cartographer.GetSosInfo( sosObjects[ i ] )
end
