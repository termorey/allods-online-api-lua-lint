На главную

avatar.GetGlobalRates()

Возвращает список глобальных рейтов развития главного игрока.

Значение рейта по умолчанию = 1. Это значит, что пользователь получает обычное количество единиц развития. Значение value больше единицы говорит о том, что игрок получает единиц развития в value раз больше.

Метод всегда возвращает значения для всех типов единиц развития. То есть, в обычных условиях будет возвращен массив из всех типом со значением value=1.

Типы единиц развития см в enum ENUM_GlobalScalerType_....

-- библиотека:
avatar

-- объявление:
function GetGlobalRates()

-- параметры:
нет

-- возвращаемые значения
table of tables - индексированный с 0 список таблиц, описывающих определённый рейт. Каждая такая таблица имеет поля:
  isCustom: boolean - определяет имеется ли у рейта кастомное описание
  type: number (enum ENUM_GlobalScalerType_...) or nil - тип рейта; nil если isCustom == true
  sysType: string (enum ENUM_GlobalScalerType_...) or nil - строковое представление типа рейта; nil если isCustom == true
  title: string or nil - заголовок рейта; nil если isCustom == false
  description: string or nil - описание рейта; nil если isCustom == false
  value: number - значение рейта
  remainingMs: number (integer) или nil - количество времени (в миллисекундах), оставшееся до окончания действия рейта; nil если время окончания неизвестно

-- пример
local globalRates = avatar.GetGlobalRates()
for i = 0, GetTableSize( globalRates ) - 1 do
  local globalRate = globalRates[ i ]
  local value = globalRate.value
end
