options.Update()

Обновляет значения всех опций по актуальной информации из системы. Сбрасывает все текущие изменения. (Базовые и новые значения выставляются согласно актуальным системным данным.)

-- библиотека:
options

-- объявление:
function Update()

-- параметры:
нет

-- возвращаемые значения:
нет

-- пример:
options.Update()
