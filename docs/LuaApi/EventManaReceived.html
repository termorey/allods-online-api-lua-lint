EVENT_MANA_RECEIVED

Событие присылается при измениние маны юнита другим юнитом.

-- поля:
unitId: ObjectId (not nil) - юнит, у которого изменилась мана
casterId: ObjectId (or nil) - юнит, который изменил ману, если информация о нём ещё доступна
addedMana: number( int ) - величина изменения маны (отрицательная - выжгли, положительная - добавили)

В зависимости от того чем была изменена мана (одно из трех):
  spellId: SpellId / nil - Id ресурса заклинания, которым изменили ману.
  abilityId: AbilityId / nil - Id абилки, если изменили абилкой.
  buffId: BuffId / nil - Id буффа, если изменили буффом.
  isFall: boolean - true, если мана изменена из-за падения
  isExploit: boolean - true, если мана изменена из-за эксплойта
