EVENT_ASTRAL_SECTOR_CHECK_JUMP_RESULT

Событие возвращающее результат проверки группы на возможность совершения прыжка в выбранный сектор астрала до выбранной POI.

-- поля:
id: ObjectId - идентификатор выбранной POI
sectorId: AstralSectorId - сектор астрала которому принадлежит выбранная POI
isSuccess: boolean - true, если прыжек доступен. Проверка завершилась успешно
failEntries: table or nil - если isSuccess == true, то nil. Список причин провала проверки (индексирован с 1), каждый элемент которого таблица с полями:
 failType: string - тип неудовлетворенного требования "ENUM_FailType_..."
 names: table - список имен аватаров (индексирован с 1) не удовлетворяющих требованию failType

См. также

  • astral.CheckJump( poiId )

  • ENUM_FailType_...
