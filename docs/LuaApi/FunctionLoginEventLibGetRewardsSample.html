loginEventLib.GetRewardsSample( day )

Получить список возможных наград за определенное количество накопленных дней посещения.

-- библиотека
loginEventLib

-- объявление:
function GetRewardsSample( day )

-- параметры
day: number (int) - текущее количество накопленных дней посещения

-- возвращаемые значения
table of ObjectId or nil - список возможных наград; индексация с 1
boolean - true если награды нужно отображать в случайном порядке

-- пример использования
local loginEventCurrencyId = avatar.GetCurrencyId( "loginEvent" )
local loginEventCurrencyValue = loginEventCurrencyId and avatar.GetCurrencyValue( loginEventCurrencyId )
local rewardsSample, randomizeView = loginEventCurrencyValue and loginEventLib.GetRewardsSample( loginEventCurrencyValue )
