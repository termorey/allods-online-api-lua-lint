SetVolume( self, volume )

Устанавливает громкость звука, диапазон 0.0f (нет звука) - 1.0f (максимальная громкость)

Внимание! Корректность установки гарантируется только для тех звуков для которых был вызван Play

-- класс
Sound2DLua

-- семантика
function SetVolume( self, volume )

-- входные параметры
self - Sound2DLua - экземпляр Sound2DLua
volume - number (float) - громкость звука, в диапазоне 0.0f - 1.0f

-- возвращаемое значение
Нет

-- пример
sound:SetVolume( 0.5 )
