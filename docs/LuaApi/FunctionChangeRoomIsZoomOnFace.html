changeRoom.IsZoomOnFace()

Возвращает true, если при выборе или создании персонажа камера назумлена на лицо.

При изменении состояния приходит EVENT_CHANGE_ROOM_ZOOM_TRIGGER_CHANGED.

-- библиотека:
changeRoom

-- объявление:
function IsZoomOnFace()

-- параметры:
нет

-- возвращаемые значения:
boolean - true, если камера назумлена на лицо

-- примеры:
if changeRoom.IsZoomEnabled() and not changeRoom.IsZoomOnFace() then
  changeRoom.ToggleZoom()
end
