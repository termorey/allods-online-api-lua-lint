loginEventLib.GetGoals()

Список ежедневных целей логин-эвента.

-- библиотека
loginEventLib

-- объявление:
function GetGoals()

-- параметры
нет

-- возвращаемые значения
table or nil - список заданий индексированный с 0 каждый элемент которого таблица с полями:
  title: WString - заголовок цели
  description: WString - описание цели
  locations: table - список ключевых локаций, каждый элемент представляет собой таблицу с полями
    locator: QuestLocation - таблица с описаниями точек цели задания

-- пример использования
if loginEventLib.IsEnabled() then
  local loginEventGoals = loginEventLib.GetGoals()
end
