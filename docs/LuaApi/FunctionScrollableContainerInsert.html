Insert( self )

Добавляет элемент в заданную позицию контейнера.

Функция вернет ошибку если выполняется любое из нижележащих условий

  • Добавляемый дочерний виджет является внутренним

  • Добавляемый дочерний виджет является формой

  • Добавляемый дочерний виджет в данный момент находится в контейнере или добавлен в сцену как Widget3D

-- класс
ScrollableContainerSafe

-- семантика
function Insert( self )

-- входные параметры:
position: Number(int) - позиция в которую добавляется элемент (счет начинается с нуля)
widget: WidgetSafe - добавляемый элемент

-- возвращаемое значение:
нет

-- пример
wtScrollableContainer:Insert( 4, wtButton ) -- вставка элемента на 5-ю позицию
