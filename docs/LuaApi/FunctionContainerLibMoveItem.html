Переместить предмет

См. ITEM_CONT_....

-- библиотека:
containerLib

-- объявление:
MoveItem( itemId, slotType, slot, count )

-- параметры:
itemId: ObjectId - идентификатор предмета
slotType: number (enum ITEM_CONT_...) - тип контейнера места назначения
slot: number (integer) or nil - номер слота в контейнере. Если указан nil, то слот подберётся автоматически
count: number (integer) or nil - количество перемещаемых предметов. Если указан nil, то переместится весь слот

-- возвращаемые значения:
нет

Пример:

containerLib.MoveItem( itemId, ITEM_CONT_INVENTORY, 12, nil )
