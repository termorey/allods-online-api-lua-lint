Текстовый виджет

TextView - это виджет для показа текстов, наследованный от базового типа Widget. С помощью него можно показать какой-либо простой или форматированный с помощью CSS текст.

В скрипте простой текст задаётся в виде объекта WString. Форматированный текст - это объект типа ValuedText, который содержит строку форматирования и подстановочные тэги, в которые можно подставлять значения типов WString, ValuedText или ValuedObject. Последние могут быть кликабельны.

Дополнительные поля текстового виджета:

  • FormatFileRef: TextFileRef - ссылка на текстовый файл. Если внутри файла правильно отформатированный XHTML документ, то текстовый виджет будет содержать ValuedText, иначе простой WString

  • TextValues: table of WidgetTextTaggedValue - список подстановочных значений по умолчанию. Каждый элемент имеет поля

    • Tag: string - название подстановочного значение

    • TagValueFileRef: TextFileRef - ссылка на текстовый файл со значением

  • DefaultTag: string - тег по-умолчанию. Используется для удобства, если формат текста подразумевает использование только одного значения подстановки.

  • TextStyle: WidgetTextStyle - стиль текста

  • minWidth: number - минимальная ширина виджета при переразмеривании. По умолчанию 0.0f;

  • maxWidth: number - максимальная ширина виджета при переразмеривании. По умолчанию 0.0f;

  • pickObjectsOnly: boolean - кликабельны только объекты ValuedObject в тексте, но не сам контрол. По умолчанию false

  • isHtmlEscaping: boolean - эскейпинг вводимого через SetVal теста. Позволяет обрабатывать ввод теста с переносами строк, предотвращает HTML-вандализм. По умолчанию false

Существующие реакции:

Текстовый виджет не имеет никаких дополнительных реакций.
