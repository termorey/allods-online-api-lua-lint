family.IsSpouse( unitId )

Проверяет является ли данный игрок супругом аватара.

-- библиотека:
family

-- объявление:
function IsSpouse( unitId )

-- параметры:
unitId: ObjectId - идентификатор игрока

-- возвращаемое значение
boolean - true если игрок супруг аватара

-- пример:
if family.IsSpouse( unitId ) then
  local spouseName = object.GetName( unitId )
end
