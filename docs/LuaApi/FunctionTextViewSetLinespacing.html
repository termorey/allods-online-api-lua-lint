SetLinespacing( self, spacing )

Устанавливает фиксированное расстояние между строками. Не рекомендуется использовать без крайней необходимости.

-- класс
TextViewSafe

-- семантика
function SetLinespacing( self, spacing )

-- входные параметры
spacing: number -- фиксированное расстояние между строками. Если 0, то подбирать размер каждой строки автоматически.

-- возвращаемое значение отсутствует

-- пример
wtTextView:SetLinespacing( 30.3 )
