EVENT_UNIT_PVP_FLAG_CHANGED

Событие посылается при изменении PvP флага какого-либо юнита (игрока или моба).

У моба может поменяться только unit.GetPvPFlagInfo(id).isOn.

-- поля:
unitId: ObjectId - Id юнита
