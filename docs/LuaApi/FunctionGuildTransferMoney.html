guild.TransferMoney()

Производит операцию с деньгами в гильдейском банке.

-- библиотека:
guild

-- объявление:
function TransferMoney( sum, changeReason, direction, comment )

-- параметры:
sum: number (integer) - сумма операции в меди
changeReason: number (enum ENUM_MoneyChangeReason_...) - причина операции
direction: number (enum ENUM_TransferMoneyOperationDirection_...) - направление перевода денег
comment: WString - комментарий при объявлении сбора

-- возвращаемое значение
нет

-- пример:
guild.TransferMoney( 10000, ENUM_MoneyChangeReason_TaxPayment, ENUM_TransferMoneyOperationDirection_Avatar2GuildBank, comment )

см. EnumMoneyChangeReason, EnumTransferMoneyOperationDirection
