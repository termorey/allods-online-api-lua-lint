avatar.GetTimeAfterRespawn()

Возвращает время (в миллисекундах), прошедшее после появления главного игрока на кладбище после вызова avatar.Respawn().

-- библиотека:
avatar

-- объявление:
function GetTimeAfterRespawn()

-- параметры:
нет

-- возвращаемые значения:
number (int) or nil - время в миллисекундах после появления игрока на кладбище. Если игрок не в состоянии воскрешения, то nil.

-- пример:
local time = avatar.GetTimeAfterRespawn()
